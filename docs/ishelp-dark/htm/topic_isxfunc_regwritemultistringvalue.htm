Программирование: RegWriteMultiStringValue

Прототип

function RegWriteMultiStringValue(const RootKey: Integer; const SubKeyName, ValueName, Data: String): Boolean;

Описание

Записывает в реестр указанное значение с типом REG_MULTI_SZ. В случае успеха функция возвращает значение True, иначе — False.

Комментарии

В значении типа REG_MULTI_SZ каждая строка разделяется нулевым символом (#0).

Пример

begin
  RegWriteMultiStringValue(HKEY_AUTO, 'Software\My Company\My Program',
    'MultiStringTest', 'String1' + #0 + 'String2' + #0 + 'String3');
end;
