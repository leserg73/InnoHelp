Программирование: RenameFile

Прототип

function RenameFile(const OldName, NewName: String): Boolean;

Описание

Выполняет попытку изменить имя файла или папки, указанного в OldFile, на новое NewFile. В случае успеха функция возвращает значение True. Если переименование не удалось (например, файл, заданный параметром NewName, уже существует), функция возвращает значение False.

Комментарии

Эта функция не может использоваться для переименования файла программы установки до тех пор, пока не начнётся процесс установки.
