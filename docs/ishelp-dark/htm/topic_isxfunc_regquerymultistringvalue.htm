Программирование: RegQueryMultiStringValue

Прототип

function RegQueryMultiStringValue(const RootKey: Integer; const SubKeyName, ValueName: String; var ResultStr: String): Boolean;

Описание

Запрашивает в указанном разделе реестра значение с типом REG_MULTI_SZ и возвращает данные в параметре ResultStr. В случае успеха функция возвращает значение True. Если функция возвращает False, то данные ResultStr остаются без изменений.

Комментарии

В значении типа REG_MULTI_SZ каждая строка разделяется нулевым символом (#0).
