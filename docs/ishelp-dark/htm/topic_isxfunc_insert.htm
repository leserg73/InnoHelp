Программирование: Insert

Прототип

procedure Insert(Source: AnyString; var Dest: AnyString; Index: Integer);

Описание

Выполняет вставку строки Source в строку Dest с указанной позиции Dest[Index].

Index — это номер символа в строке. Нумерация начинается с 1. Если Index меньше 1, вставка происходит в начале строки, как если бы было указано значение 1. Если значение Index больше длины строки, то к строке Dest добавляется строка Source.
