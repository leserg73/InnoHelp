Программирование: GetMD5OfString

Прототип

function GetMD5OfString(const S: AnsiString): String;

Описание

Возвращает строковое значение контрольной суммы MD5 для заданной строки в кодировке ANSI.

Пример

var
  MD5: String;
begin
  MD5 := GetMD5OfString('Test');
  // MD5 = '0cbc6611f5540bd0809a388dc95a615b'
end;
