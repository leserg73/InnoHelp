[Setup]: SignToolRunMinimized

Допустимые значения:
yes или no

Значение по умолчанию:
no

Описание

Если указано значение yes, компилятор будет запускать инструмент цифровой подписи в свёрнутом окне. Не имеет никакого эффекта, если инструмент цифровой подписи является консольным приложением.
