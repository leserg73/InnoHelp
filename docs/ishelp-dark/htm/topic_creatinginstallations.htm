Создание инсталляторов

Инсталляторы создаются посредством сценариев, которые представляют собой обычные текстовые файлы в кодировке ASCII или Unicode (UTF-8) и форматом, несколько похожим на файлы INI. (Нет, сценарии не настолько сложные, как можно было бы подумать!)

Сценарии имеют расширение ".iss" (сокращение от Inno Setup Script). Сценарий управляет каждым аспектом установки. Он определяет, какой файл и куда следует установить, какие ярлыки следует создать и как они будут называться, и так далее.

Файлы сценариев обычно редактируются в интегрированной среде разработки Inno Setup (англ. Integrated Development Environment; далее по тексту: IDE компилятора или просто компилятор). Следующий и заключительный шаг после составления сценария — это его компиляция. Для этого в IDE компилятора нужно выбрать "Компилировать". В результате на основе сценария будет создана готовая к запуску программа установки. По умолчанию файл инсталлятора создаётся в папке с именем "Output", которая находится в папке сценария.

Чтобы получить общее представление о том как это все работает, запустите IDE компилятора, нажмите Файл | Открыть и выберите один из примеров сценария, которые находятся в папке "Examples" по месту установки Inno Setup. (Примеры сценариев можно использовать в качестве шаблонов для создания собственных сценариев.)
