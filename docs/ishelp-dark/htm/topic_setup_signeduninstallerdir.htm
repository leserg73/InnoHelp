[Setup]: SignedUninstallerDir

Значение по умолчанию:
OutputDir

Описание

Определяет каталог, в котором должны быть созданы файлы подписанного деинсталлятора. По умолчанию такие файлы сохраняются в каталоге вывода.

Разные файлы сценариев могут использовать в директиве SignedUninstallerDir одно и то же значение. Если используется общий каталог для создания файлов подписанного деинсталлятора, нет необходимости в повторном подписывании деинсталлятора при компиляции сценариев с различным значением OutputDir.

Если SignedUninstallerDir не содержит полного пути, то он будет создан относительно пути в директиве SourceDir. Установка в директиве SignedUninstallerDir значения . приведёт к выводу файлов деинсталлятора в исходный каталог сценария.

Пример:
SignedUninstallerDir=c:\signeduninstallers
