Общие параметры

Существует три необязательных параметра, поддерживаемых всеми секциями, записи которых содержат перечисляемые параметры. Вот они:

Languages

Список поддерживаемых языков, разделенных пробелами, которые указывают программе установки, к каким языкам относится запись. Если конечный пользователь выберет язык из этого списка, то такая запись будет обработана (например, выполнится установка файла).

Запись без параметра Languages обрабатывается всегда, пока другие параметры этого не отменят.

Помимо списков, разделённых пробелами, также можно использовать логические выражения, содержащих названия языков. Смотрите раздел справки "Параметры секций [Components] и [Tasks]" с примерами логических выражений.

Пример:
Languages: en nl
MinVersion

Определяет минимальную версию Windows для обработки записи. Если использовать значение 0, то эта запись никогда не будет обработана. Значение также может содержать номер сборки и версию пакета обновления. Переопределяет любое значение директивы MinVersion в секции [Setup].

Запись без параметра MinVersion обрабатывается всегда, пока другие параметры этого не отменят.

Пример:
MinVersion: 6.2
OnlyBelowVersion

Является противоположностью параметра MinVersion. Указывает минимальную версию Windows, при которой запись не будет обработана. Например, если вы задали значение 6.2 и пользователь запустил программу установки по управлением Windows 7, то эта запись будет обработана, но при запуске на системе Windows 8 (версия 6.2) или выше, эта запись обработана не будет. Установка значения 0 снимает какие-либо ограничения на версию. Значение также может содержать номер сборки и версию пакета обновления. Переопределяет любое значение директивы OnlyBelowVersion в секции [Setup].

Если значение 0 не используется, следует указывать значение больше, чем 6.1, иначе запись не будет обработана. (ОС Windows Vista/Server 2008 больше не поддерживаются.)

Запись без параметра OnlyBelowVersion обрабатывается всегда, пока другие параметры этого не отменят.

Пример:
OnlyBelowVersion: 6.2
