[Setup]: AppPublisher

Описание

Определяет сведения о приложении для поля "Издатель" в апплете Панели управления "Удаление или изменение программы". Значение директивы может содержать константы.

Значение этой директивы также используется в качестве значения по умолчанию для директивы VersionInfoCompany, если оно не указано.

Пример:
AppPublisher=My Company, Inc.
AppPublisherURL=http://www.example.com/
