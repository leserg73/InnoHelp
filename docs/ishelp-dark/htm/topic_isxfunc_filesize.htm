Программирование: FileSize

Прототип

function FileSize(const Name: String; var Size: Integer): Boolean;

Описание

Устанавливает заданный размер Size (в байтах) для указанного файла Name. Возвращает значение True, если размер был успешно установлен, иначе False. Поддерживаются размеры файлов не более 2Гб. Чтобы установить 64-разрядный размер файла, вместо этой функции используйте FileSize64.
