Программирование: RaiseException

Прототип

procedure RaiseException(const Msg: String);

Описание

Выполняет вызов исключения с указанным сообщением Msg.

Пример

begin
  RaiseException('Сообщение об исключительной ситуации...');

  // Следующая строка не будет выполнена из-за вызова исключения
  MsgBox('Это сообщение никто не увидит!', mbInformation, MB_OK);
end;
