Программирование: LogFmt

Прототип

procedure LogFmt(const S: String; const Args: array of const);

Описание

Форматирует строку S последовательностью аргументов из массива Args и записывает в файл журнала.

Пример

function OnDownloadProgress(const Url, FileName: String; const Progress, ProgressMax: Int64): Boolean; 
begin
  LogFmt('%s->%s: %d/%d', [Url, FileName, Progress, ProgressMax]);
  Result := True;
end;
