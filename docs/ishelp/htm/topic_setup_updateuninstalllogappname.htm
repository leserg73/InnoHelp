[Setup]: UpdateUninstallLogAppName

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

При значении yes существующий журнал деинсталляции будет дополнен, а поле AppName заменено текущим значением AppName программы установки. Поле AppName в журнале деинсталляции определяет заголовок, который будет показывать деинсталлятор. Если установка представляет собой обычное обновление или дополнение к существующей программе и нет никакой необходимости изменять название деинсталлятора, используйте значение no.
