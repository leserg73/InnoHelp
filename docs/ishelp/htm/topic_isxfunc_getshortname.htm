Программирование: GetShortName

Прототип

function GetShortName(const LongName: String): String;

Описание

Возвращает краткое имя указанного файла согласно формату именования версии 8.3 (не более 8-ми символов). Если краткое имя не найдено, то функция возвращает исходное имя файла, заданное параметром LongName.
