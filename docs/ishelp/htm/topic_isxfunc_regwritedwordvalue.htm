Программирование: RegWriteDWordValue

Прототип

function RegWriteDWordValue(const RootKey: Integer; const SubKeyName, ValueName: String; const Data: Cardinal): Boolean;

Описание

Записывает в реестр указанное значение с типом REG_DWORD. В случае успеха функция возвращает значение True, иначе — False.

Пример

begin
  RegWriteDWordValue(HKEY_AUTO, 'Software\My Company\My Program',
    'ShowToolbar', 1);
end;
