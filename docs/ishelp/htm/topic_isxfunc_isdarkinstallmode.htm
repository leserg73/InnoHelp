Программирование: IsDarkInstallMode

Прототип

function IsDarkInstallMode: Boolean;

Описание

Возвращает значение True, если инсталлятор или деинсталлятор запущен с использованием тёмного стиля.

Комментарии

Может возвращать значение True, даже если IsWinDark возвращает False.

Может не возвращать значение True, если HighContrastActive возвращает True.

IsWinDark
HighContrastActive
