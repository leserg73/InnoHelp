[Setup]: WizardSmallImageBackColor

Допустимые значения:
Цвет в формате #rrggbb или $bbggrr, где bb, gg и rr определяют интенсивности (в шестнадцатеричном формате) для синего, зеленого и красного цветов соответственно. Или это может быть одно из следующих предопределённых названий цветов: black, maroon, green, olive, navy, purple, teal, gray, silver, red, lime, yellow, blue, fuchsia, aqua, white, none, window.

Значение по умолчанию:
(см. ниже)

Описание

Эта директива устанавливает цвет фона, используемого для заполнения всех неиспользуемых или прозрачных областей в малом изображении мастера установки, которое задаётся директивой WizardSmallImageFile. Использование значения none в качестве цвета заливки сохраняет прозрачность.

Если используется фоновое изображение, указанное с помощью директивы WizardBackImageFile, то по умолчанию для этой директивы устанавливается значение none.

Если же для WizardSmallImageFile указано пустое значение, то по умолчанию также принимается значение none.

Если используется пользовательское изображение мастера, то по умолчанию эта директива принимает значение window (обычно белый).

Во всех остальных случаях по умолчанию используется значение none.
