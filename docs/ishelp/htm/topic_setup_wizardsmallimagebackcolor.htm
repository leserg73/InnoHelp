[Setup]: WizardSmallImageBackColor

Допустимые значения:
Цвет в формате #rrggbb или $bbggrr, где bb, gg и rr определяют интенсивности (в шестнадцатеричном формате) для синего, зеленого и красного цветов соответственно. Или это может быть одно из следующих предопределённых названий цветов: clBlack, clMaroon, clGreen, clOlive, clNavy, clPurple, clTeal, clGray, clSilver, clRed, clLime, clYellow, clBlue, clFuchsia, clAqua, clWhite, clNone, clWindow.

Значение по умолчанию:
clNone, если используется встроенное изображение мастера, в противном случае — clWindow.

Описание

Эта директива устанавливает цвет фона, используемого для заполнения всех неиспользуемых или прозрачных областей в малом изображении мастера установки, которое задаётся директивой WizardSmallImageFile. Использование значения clNone в качестве цвета заливки сохраняет прозрачность.
