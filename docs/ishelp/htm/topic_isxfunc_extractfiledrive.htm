Программирование: ExtractFileDrive

Прототип

function ExtractFileDrive(const FileName: String): String;

Описание

Возвращает строку с диском для полного пути к файлу, переданного в параметре FileName. Для локальных путей с буквой диска итоговая строка будет представлена по форме '<буква диска>:' (напр. С:). Для путей в формате UNC итоговая строка будет иметь вид '\\<servername>\<sharename>'. Если заданный путь не содержит ни одного определения диска, результатом функции будет пустая строка.
