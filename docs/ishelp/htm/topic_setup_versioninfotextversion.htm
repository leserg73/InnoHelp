[Setup]: VersionInfoTextVersion

Значение по умолчанию:
VersionInfoVersion

Описание

Определяет текстовое значение версии файла для сведений о версии программы установки.

Обратите внимание, что это значение отображалось только на вкладке версии проводника в Windows 98 и более ранних. Более поздние версии Windows отображают двоичное значение версии (VersionInfoVersion).

Эта директива не учитывается, если UseSetupLdr имеет значение no.
