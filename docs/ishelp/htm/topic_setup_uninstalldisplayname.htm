[Setup]: UninstallDisplayName

Описание

Эта директива позволяет указать другое имя приложения, которое будет отображаться в апплете панели управления "Удаление или изменение программы". Значение может содержать константы. Если директива не задана или пустая, программа установки будет использовать значение AppVerName секции [Setup].

Пример:
UninstallDisplayName=My Program
