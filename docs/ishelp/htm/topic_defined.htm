Препроцессор Inno Setup: Defined

Прототип функции

int Defined(<ident>)
int Defined <ident>

Описание

Функция является специальной. Возвращает нулевое значение, если указанный идентификатор (переменная) НЕ определён директивой define.

Использовать круглые скобки для указания идентификатора необязательно.
