[Setup]: WizardImageBackColorDynamicDark

Значение по умолчанию:
#314853 (голубовато-тёмно-серый)

Описание

Переопределяет значение директивы WizardImageBackColor при использовании WizardStyle со значением dynamic и тёмном режиме Windows. В остальных случаях игнорируется.

Более подробную информацию смотрите в описании WizardImageBackColor.
