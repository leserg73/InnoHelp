Программирование: GetVersionNumbersString

Прототип

function GetVersionNumbersString(const Filename: String; var Version: String): Boolean;

Описание

Выполняет чтение номера версий из указанного в параметре Filename файла. В случае успеха функция возвращает значение True, иначе — False. Полученное значения номера версии передаётся в параметре Version и представлено в формате "0.0.0.0". Значение версии берётся из информации о версии файла и соответствует полю "File version".
