[Setup]: AppVerName

Значение по умолчанию:
AppName версия AppVersion, локализованное в соответствии с пользовательским сообщением NameAndVersion для активного языка

Описание

Эта директива определяет имя и версию устанавливаемого приложения. Значение этого параметра отображается на странице "Приветствие" программы установки и используется в качестве названия по умолчанию для приложений в апплете "Удаление или изменение программы" (смотрите директиву UninstallDisplayName). Значение может содержать константы.

Если директиве DisableWelcomePage задано значение yes (является значением по умолчанию), то в заголовках окон используется значение директивы AppVerName вместо AppName.

Эта директива является обязательной, если директива AppVersion не используется.

Примеры:
AppVerName=My Program 1.5
AppVerName=My Program version 1.5
AppVerName={cm:NameAndVersion,My Program,1.5}
