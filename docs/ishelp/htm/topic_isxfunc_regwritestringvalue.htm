Программирование: RegWriteStringValue

Прототип

function RegWriteStringValue(const RootKey: Integer; const SubKeyName, ValueName, Data: String): Boolean;

Описание

Записывает в реестр указанное значение с типом REG_SZ. В случае успеха функция возвращает значение True, иначе — False.

Комментарии

Если значение уже существует и имеет тип REG_EXPAND_SZ, новое значение также будет иметь тип REG_EXPAND_SZ. В противном случае будет создано значение с типом REG_SZ.

Пример

begin
  RegWriteStringValue(HKEY_AUTO, 'Software\My Company\My Program',
    'Language', ExpandConstant('{language}'));
end;
