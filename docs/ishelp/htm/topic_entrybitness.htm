Разрядность записи

В таблице ниже перечислены секции, в которых записи могут быть либо "32-бит", либо "64-бит".

По умолчанию разрядность записей соответствует режиму установки установщика и программы удаления: 32-бит в 32-разрядном режиме установки и 64-бит в 64-разрядном режиме установки. При этом для каждой записи возможна индивидуальная установка разрядности.

Секция На что влияет разрядность записи Как переопределить
[Dirs] Парметр Name: перенаправление системного пути. Невозможно
[Files]
  • Параметры Source и DestDir: перенаправление системного пути.
  • Флаг regserver: каким файлом regsvr32.exe регистрируется файл.
  • Флаг regtypelib: в каком представлении реестра регистрируется библиотека типов.
  • Флаг sharedfile: в каком представлении реестра обновляется ключ SharedDLLs.
 Использовать флаг 32bit или 64bit.
[Icons] Флаг useapppaths: к какому представлению реестра осуществляется доступ. Невозможно
[InstallDelete]
[UninstallDelete]		 Параметр Name: перенаправление системного пути. Невозможно
[Registry] К какому представлению реестра осуществляется доступ. Для параметров Root использовать суффикс 32 или 64.
[Run]
[UninstallRun]		 Параметры Filename и WorkingDir: перенаправление системного пути. Использовать флаг 32bit или 64bit.
