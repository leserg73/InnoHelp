Программирование: MinimizePathName

Прототип

function MinimizePathName(const Filename: String; const Font: TFont; MaxLen: Integer): String;

Описание

Возвращает сокращенное имя указанного файла Font в соответствии с указанным шрифтом Font и максимальным размером отображения MaxLen в пикселях. Отброшенная часть строки будет заменена знаком многоточия (...).

Пример

MyLabel.Caption := MinimizePathName(MyPathName, MyLabel.Font, MyLabel.Width);
