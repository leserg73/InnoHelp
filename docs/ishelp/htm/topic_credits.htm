Разработчики

Программа Inno Setup была создана Jordan Russell и в настоящее время (начиная с выпуска 5.4.3 в 2011 году) поддерживается Martijn Laan.

Ниже представлен список лиц, кто внёс значительный вклад в развитие и популяризацию проекта Inno Setup, а также заслуживает особого признания:

  • Jean-Loup Gailly и Mark Adler: создатели библиотеки zlib для сжатия данных, которая используется Inno Setup.
  • Julian Seward: создатель библиотеки bzlib для сжатия данных, которая используется Inno Setup.
  • Igor Pavlov: создатель библиотек 7-Zip и LZMA SDK, которые используется в Inno Setup.
  • Vince Valenti: большая часть кода для директив "Window" в секции [Setup] (1.12.4).
  • Joe White: код для директивы ChangesAssociations в секции [Setup] (1.2).
  • Jason Olsen: большая часть кода для обновления журнала деинсталлятора (1.3).
  • Martijn Laan: поддержка Rich Edit 2.0 и URL-детектор (1.3.13); тихая деинсталляция (1.3.25); поддержка системных значков в списках дисков и каталогов (1.3.25); тихая установка (2.0); секции [Types], [Components] и [Tasks] (2.0); флаг postinstall (2.0); секция [Code] (4.0); поддержка вложенных компонентов и задач (4.0); многие другие функции после 4.0.
  • Alex Yackimoff: участие в создании компонента TNewCheckListBox (4.0); ISPP (5.4.1).
  • Carlo Kok: библиотека RemObjects Pascal Script (4.0).
  • Создатели SynEdit: редактор с синтаксической подсветкой кода, используемый IDE компилятора (2.0 - 5.2.4).
  • Создатели UniSynEdit: редактор с синтаксической подсветкой кода, используемый в IDE компилятора (5.3 - 5.3.11).
  • Создатели Scintilla: редактор с синтаксической подсветкой кода, используемый в IDE компилятора (5.4).
  • Zaher Dirkey: начальная работа над улучшенной поддержкой языков справа налево (5.2.3).
  • Evgeny Karpov из RemObjects Software: начальная поддержка Unicode версии (5.3).
  • Motaz Alnuweiri: иконки размером 128x128 и 256x256 для IDE компилятора и документов сценариев (5.5.3).
  • DRON: код улучшенного растяжения изображений (5.6).
  • Sherlock Software: основная часть кода для поддержки функции CreateCallback (6.0).
  • Vizit0r: код просмотра "Стека вызовов" (6.0.3).
  • Cristoph Nahr: код определения версии .NET (6.0.4).
  • Sergii Leonov: код дизайнера MsgBox (6.1.0) и начальная работа над инструментом импорта реестра (6.3.0) для IDE компилятора; перевод справки Inno Setup и Препроцессор Inno Setup, а также создание онлайн версии справки на портале GitHub.
  • Jens Geyer: начальная работа по закрытию вкладок в IDE компилятора (6.3.0).
  • ser163: код инструмента по добавлению файлов и папок в IDE компилятора (6.3.0).

Огромная благодарность всем, кто отвечает на вопросы пользователей форума и портала Stack Overflow, особенно Gavin Lambert, Deanna Hants, Jernej Simoncic, Martin Prikryl и TLama.
