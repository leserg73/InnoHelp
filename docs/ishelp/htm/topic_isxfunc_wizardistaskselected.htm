Программирование: WizardIsTaskSelected

Прототип

function WizardIsTaskSelected(const Tasks: String): Boolean;

Описание

Возвращает значение True, если выбрана указанная задача Tasks. Несколько задач можно указать таким же образом, как и параметры Tasks.

Пример

begin
  if WizardIsTaskSelected('desktopicon') then
    // задача 'desktopicon' успешно выбрана
end;
