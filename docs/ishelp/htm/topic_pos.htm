Препроцессор Inno Setup: Pos

Прототип функции

int Pos(str SubStr, str S)

Описание

Выполняет поиск подстроки SubStr в строке S и возвращает целочисленное значение (начиная с 1), которое является позицией первого символа подстроки SubStr в строке S. Если подстрока SubStr в строке S не найдена, функция возвращает нулевое значение.
