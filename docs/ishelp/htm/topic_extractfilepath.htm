Препроцессор Inno Setup: ExtractFilePath

Прототип функции

str ExtractFilePath(str PathName)

Описание

Выполняет извлечение пути к папке для заданного файла, включая завершающий символ обратной косой черты. Если PathName не содержит путь к файлу, то функция возвращает пустую строку.

Функция объявлена в файле ISPPBuiltins.iss.
