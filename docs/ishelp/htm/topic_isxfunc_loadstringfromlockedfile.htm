Программирование: LoadStringFromLockedFile

Прототип

function LoadStringFromLockedFile(const FileName: String; var S: AnsiString): Boolean;

Описание

Загружает указанный двоичный или текстовый файл FileName в кодировке ANSI в заданную строку S, даже если файл уже открыт для записи в другой программе. В случае успеха функция возвращает значение True, иначе — False.
