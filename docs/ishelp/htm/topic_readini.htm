Препроцессор Inno Setup: ReadIni

Прототип функции

str ReadIni(str 1, str 2, str 3, str? 4)

Описание

Выполняет чтение значений из INI-файла. Первый параметр является именем INI-файла. Второй параметр — это имя секции в заданном INI-файле. Третий параметр — имя ключа, значение которого необходимо получить. Последний параметр (4) является необязательным и может быть использован для установки значения, которое вернёт функция в случае неудачного выполнения операции или при пустом значении запрошенного ключа.
