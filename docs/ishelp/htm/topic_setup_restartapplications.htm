[Setup]: RestartApplications

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

Если указано значение yes и директива CloseApplications также имеет значение yes, программа установки перезапустит закрытое приложение после завершения установки.

Примечание: чтобы программа установки смогла перезапустить приложение после завершения установки, приложение должно использовать API-функцию Windows RegisterApplicationRestart.
