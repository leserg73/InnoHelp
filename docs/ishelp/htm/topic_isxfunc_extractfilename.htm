Программирование: ExtractFileName

Прототип

function ExtractFileName(const FileName: String): String;

Описание

Извлекает имя и расширение для указанного имени файла, переданного в параметре FileName. Итоговая строка будет содержать все символы правой части FileName, начиная с первого символа после двоеточия или обратной косой черты, которые отделяют имя пути от имени файла с расширением. Если заданное имя не содержит определение пути (диск и каталог), то результатом будет исходное имя файла.
