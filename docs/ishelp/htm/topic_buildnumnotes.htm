Использование номера сборки и/или пакета обновления

Версии системы, определённые в MinVersion и OnlyBelowVersion, могут также содержать номера сборки и/или версии пакета обновления.

Примеры:
6.1sp1
10.0.22000

Если номер сборки не указан или равен нулю, то программа установки не будет проверять номер сборки системы.

Если версия пакета обновления не указана или равна нулю, то программа установки не будет проверять версию пакета обновления системы.

Если версия пакета обновления указана, то программа установки будет сравнивать её с версией пакета обновления системы только в том случае, если указанный номер старшей и младшей версии полностью соответствует версии системы. Например, если для MinVersion указано значение 6.1sp1, то программа установки выполнит проверку только для SP1 на системах Windows 7 и Windows Server 2008 R2 (6.1).

В параметре OnlyBelowVersion: если указанная версия совпадает с версией системы, то обычно программа установки считает эту версию слишком высокой. Однако, если ещё указана и версия пакета обновления, то указанная версия может оказаться в пределах допуска версии системы.
