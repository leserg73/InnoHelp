Программирование: CreateCustomPage

Прототип

function CreateCustomPage(const AfterID: Integer; const ACaption, ADescription: String): TWizardPage;

Описание

Выполняет создание пользовательской страницы для мастера установки. По умолчанию страница будет пустой, поэтому вам нужно будет ещё создать собственные элементы управления и поместить их на страницу (установив свойство Parent в свойство Surface для экземпляра TWizardPage, возвращаемого этой функцией).

Параметр AfterID определяет страницу мастера установки, после которой будет размещена (показана) эта страница.

Для параметра ACaption следует указать заголовок страницы, а для параметра ADescription краткое описание назначения страницы.

Пример

Смотрите файл CodeClasses.iss и AllPagesExample.iss из папки "Examples" в каталоге установки Inno Setup.
