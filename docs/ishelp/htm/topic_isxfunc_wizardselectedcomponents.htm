Программирование: WizardSelectedComponents

Прототип

function WizardSelectedComponents(const Descriptions: Boolean): String;

Описание

Возвращает разделенный запятыми перечень имен или описаний компонентов, выбранных пользователем. Если параметр Descriptions имеет значение True, то функция вернёт список описаний, в противном случае — имена.
