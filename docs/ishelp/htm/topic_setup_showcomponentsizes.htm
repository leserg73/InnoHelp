[Setup]: ShowComponentSizes

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

При значении yes программа установки будет показывать размеры компонентов в списке компонентов. В зависимости от размера наибольшего из компонентов программа установка будет показывать размеры в килобайтах или мегабайтах.
