[Setup]: UsePreviousGroup

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

При значении yes программа установки во время своего запуска выполнит в реестре поиск информации об установке того же приложения и, если таковая будет найдена, использует имя папки в меню Пуск от предыдущей установки в качестве папки по умолчанию и предложит её пользователю. Кроме того, если параметру AllowNoIcons задано значение yes, будет восстановлена опция "Не создавать папку в меню «Пуск»" из предыдущей установки.

Обратите внимание, что программа установки не может повторно использовать настройки предыдущей установки, у которой директива Uninstallable имела значение no, поскольку искомые записи реестра, когда Uninstallable=no, не создаются.
