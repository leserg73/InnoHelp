[Setup]: DefaultDialogFontName

Значение по умолчанию:
Tahoma

Описание

Определяет имя шрифта, который должен использоваться в диалогах для языков, у которых не задан или является пустым параметр DialogFontName в секции [LangOptions].

Если указанный шрифт в системе пользователя не существует или задана пустая строка, будет выполнена подмена на шрифт Microsoft Sans Serif или MS Sans Serif размером 8 пунктов.
