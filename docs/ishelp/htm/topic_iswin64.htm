Препроцессор Inno Setup: IsWin64

Прототип функции

int IsWin64()

Описание

Возвращает ненулевое значение, если система работает под управлением 64-бит версии Windows, которая обеспечивает необходимую поддержку API Inno Setup для выполнения 64-разрядных задач установки. В противном случае функция возвращает нулевое значение.
