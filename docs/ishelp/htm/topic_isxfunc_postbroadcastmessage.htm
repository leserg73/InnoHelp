Программирование: PostBroadcastMessage

Прототип

function PostBroadcastMessage(const Msg, WParam, LParam: Longint): Boolean;

Описание

Посылает (вставляет в очередь) указанное сообщение Msg в окно верхнего уровня системы и немедленно возвращает результат операции.

Указанное сообщение должно быть уникальным. Используйте функцию RegisterWindowMessage для получения таких сообщений.
