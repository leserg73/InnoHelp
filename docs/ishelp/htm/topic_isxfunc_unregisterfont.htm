Программирование: UnregisterFont

Прототип

procedure UnregisterFont(const FontName, FontFilename: String; const PerUserFont: Boolean);

Описание

Выполняет отмену регистрации шрифта с указанным именем FontName и именем файла FontFilename. С помощью параметра PerUserFont можно задать временную отмену регистрации шрифта (до перезагрузки ПК), указав значение False, или на постоянной основе (навсегда), указав значение True.
