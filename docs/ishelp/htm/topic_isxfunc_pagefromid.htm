Программирование: PageFromID

Прототип

function PageFromID(const ID: Integer): TWizardPage;

Описание

По указанному ID страницы функция возвращает её экземпляр класса TWizardPage. Используйте эту функцию, если, к примеру, хотите получить доступ к свойствам страницы, зная только её ID.

Если указан неверный ID страницы, будет вызвано исключение.

Пример

var
  Page: TWizardPage;
begin
  Page := PageFromID(wpWelcome);
  Page.Surface.Color := clBlue;
end;
