Программирование: RegisterTypeLibrary

Прототип

procedure RegisterTypeLibrary(const Is64Bit: Boolean; const Filename: String);

Описание

Регистрирует библиотеку типов с указанным именем файла Filename. В случае неудачи будет вызвано исключение.

Если для параметра Is64Bit указано значение True, то файл библиотеки будет зарегистрирован в 64-разрядном представлении реестра. Если же указано значение False — в 32-разрядном представлении реестра. В Inno Setup версии 7.0 и выше этот параметр должен совпадать с разрядностью программы установки/удаления, поэтому необходимо передавать значение IsCurrentProcess64Bit.

Пример

begin
  // Регистрация stdole2.tlb, находящейся в каталоге System.
  // - Если SetupArchitecture=x64, регистрация будет выполнена в 64-разрядном представлении реестра.
  // - Если SetupArchitecture=x86, регистрация будет выполнена в 32-разрядном представлении реестра.
  RegisterTypeLibrary(IsCurrentProcess64Bit, ExpandConstant('{sys}\stdole2.tlb'));
end;
