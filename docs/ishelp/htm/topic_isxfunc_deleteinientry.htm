Программирование: DeleteIniEntry

Прототип

procedure DeleteIniEntry(const Section, Key, Filename: String);

Описание

Выполняет удаление указанного ключа из файла INI.

Параметр FileName определяет имя файла INI, в котором следует удалить ключ. Параметром Section задаётся имя раздела, а параметром Key — имя ключа.
