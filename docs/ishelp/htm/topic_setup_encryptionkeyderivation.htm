[Setup]: EncryptionKeyDerivation

Допустимые значения:
pbkdf2
или от pbkdf2/1 до pbkdf2/2147483647

Значение по умолчанию:
pbkdf2/220000

Описание

Здесь указывается функция формирования ключа, которая будет использоваться для получения ключа шифрования на основе пароля, указанного в директиве Password, и заданном количестве итераций.

pbkdf2 – это функция шифрования PBKDF2-HMAC-SHA256 с 128-разрядным псевдослучайным значением соли, которая для дополнительной безопасности позволяет увеличить количество её итераций. Если число итераций не указано, то по умолчанию принимается значение 220000.
