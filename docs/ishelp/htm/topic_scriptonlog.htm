Программирование: Параметр "OnLog"

В секции [Run] поддерживается один дополнительный параметр. Этот параметр следующий:

OnLog

В сочетании с флагом logoutput этот параметр определяет имя функции, которая будет вызываться для каждой строки вывода программы вместо непосредственного включения её в файл журнала.

Сама функция должна быть объявлена в секции [Code] и прототип её следующий:
    procedure MyOnLog(const S: String; const Error, FirstLine: Boolean);.

См. описание функции ExecAndLogOutput, где поясняются параметры S, Error и FirstLine.

Пример:
[Run]
Filename: "{app}\MyConsoleProg.exe"; OnLog: MyOnLog; Flags: logoutput

Ниже показан пример секции [Code], в котором приведена используемая выше процедура. Функция Log является вспомогательной и не включена в эту секцию [Code].

[Code]
procedure MyOnLog(const S: String; const Error, FirstLine: Boolean);
begin
  if not Error and FirstLine then
    Log('Output:');
  Log(S);
end;
