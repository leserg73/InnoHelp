Препроцессор Inno Setup: Разделение строк

Если в сценарии содержатся длинные строки, то при использовании команды ISPP их можно разделить на более короткие. Это очень удобно и избавит вас от лишних движений при навигации по тексту сценария. Для разделения строк используется символ "\" (обратная косая черта), которому предшествует символ пробела. При синтаксическом анализе сценария, препроцессор, встретив в конце строки символ "\" будет знать, что данная строка неполная и её продолжение находится на следующей строке и т.д. Эти строки, в конечном итоге, будут обработаны как одна. Например:

#define MyAppName \
  "Управление базами данных SQL Server Compact Edition"

Обратите внимание, что перед символом разделения строк находится символ пробела. В обычном виде указанная команда сценария была бы записана так:

#define MyAppName "Управление базами данных SQL Server Compact Edition"

Используемый по умолчанию символ разделения строк "\" может быть изменён при помощи директивы pragma.
