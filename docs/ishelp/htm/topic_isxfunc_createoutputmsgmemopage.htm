Программирование: CreateOutputMsgMemoPage

Прототип

function CreateOutputMsgMemoPage(const AfterID: Integer; const ACaption, ADescription, ASubCaption: String; const AMsg: AnsiString): TOutputMsgMemoWizardPage;

Описание

Выполняет создание на странице мастера установки определённого текста, доступного только для чтения, а также многострочного текстового поля, способного отобразить документ в формате RTF.

Параметр AfterID определяет страницу мастера установки, после которой будет размещена (показана) эта страница.

Параметр ACaption определяет текст для заголовка страницы, а параметр ADescription — краткое описание назначения страницы.

Параметр ASubCaption определяет неизменный текст для отображения, а параметр AMsg — текст, который будет назначен элементу текстового поля.

Пример

var
  Page: TOutputMsgMemoWizardPage;

...

// Создаём страницу
Page := CreateOutputMsgMemoPage(wpWelcome,
  'Информация', 'Пожалуйста, ознакомьтесь с важной информацией.',
  'Когда вы будете готовы продолжить установку, нажмите "Далее".',
  'Бла-бла-бла.');

Смотрите другой пример в файле AllPagesExample.iss из папки "Examples" в каталоге установки Inno Setup.
