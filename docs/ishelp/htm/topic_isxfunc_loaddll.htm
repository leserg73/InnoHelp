Программирование: LoadDLL

Прототип

function LoadDLL(const DLLName: String; var ErrorCode: Integer): Longint;

Описание

Выполняет загрузку указанного файла DLL, указанного в параметре DLLName. Если файл был успешно загружен, то возвращает дескриптор DLL, в противном случае ноль. Если функция вернула ноль, то параметр ErrorCode укажет на причину ошибки. Для получения описания ошибки используйте вызов SysErrorMessage(ErrorCode).

Комментарии

Эта функция устарела. Смотрите раздел Использование DLL.
