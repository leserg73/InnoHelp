Программирование: SaveStringsToUTF8File

Прототип

function SaveStringsToUTF8File(const FileName: String; const S: TArrayOfString; const Append: Boolean): Boolean;

Описание

Сохраняет указанный массив строк TArrayOfString в заданном файле FileName в кодировке UTF-8 с маркером BOM. Если параметр Append имеет значение True и заданный файл уже существует, то строки будут добавлены к файлу, вместо его полной перезаписи. В случае успеха возвращает значение True, иначе — False.
