Препроцессор Inno Setup: WriteIni

Прототип функции

void WriteIni(str FileName, str Section, str Key, any Value)

Описание

Выполняет запись указанного значения в INI-файл. Параметр FileName является именем INI-файла. Section — это имя секции в INI-файле. Key — имя ключа в указанной секции. Последний параметр Value должен содержать значение (любого типа), которое необходимо записать в INI-файл.
