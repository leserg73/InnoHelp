Программирование: WizardIsComponentSelected

Прототип

function WizardIsComponentSelected(const Components: String): Boolean;

Описание

Возвращает значение True, если выбран заданный компонент Components. Несколько компонентов можно указать таким же образом, как и параметры Components.

Пример

begin
  if WizardIsComponentSelected('helpfiles') then
    // компонент 'helpfiles' успешно выбран
end;
