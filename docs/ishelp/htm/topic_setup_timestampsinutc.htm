[Setup]: TimeStampsInUTC

Допустимые значения:
yes или no

Значение по умолчанию:
no

Описание

По умолчанию отметки времени файлов секции [Files], для которых не указан флаг external, сохраняются и восстанавливаются в виде локального времени. Это означает, что если определённый файл во время компиляции получил отметку 01:00 по местному времени, он будет извлечён программой установки с отметкой 01:00 по местному времени, независимо от настроек часового пояса пользователя или действия DST.

Если директива TimeStampsInUTC имеет значение yes, отметка времени будет сохранена и восстановлена в формате UTC — собственном формате времени Win32 и NTFS. В этом режиме файл с отметкой времени 01:00 по местному времени в Нью-Йорке будет иметь отметку времени 06:00 по местному времени при установке в Лондоне.
