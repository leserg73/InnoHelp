[Setup]: SlicesPerDisk

Допустимые значения:
от 1 до 26

Значение по умолчанию:
1

Описание

Определяет количество файлов Setup-*.bin, создаваемых для каждого диска. При значении 1 (по умолчанию) файлы будут именоваться в формате Setup-x.bin, где x – это номер диска. Если значение больше 1, файлы будут именоваться в формате Setup-xy.bin, где x – номер диска, а y – уникальная буква.

Одна из причин, по которой может потребоваться увеличить значение по умолчанию, заключается в том, что размер дискового носителя превышает 2100000000 байт — верхний предел директивы DiskSliceSize секции [Setup]. Если, к примеру, объём дисковых накопителей составляет 3000000000 байт, можно преодолеть ограничение в 2100000000 байт, указав для SlicesPerDisk значение 2, а для DiskSliceSize значение 1500000000 (или чуть меньше из-за служебных данных файловой системы).
