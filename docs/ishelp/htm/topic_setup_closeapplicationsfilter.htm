[Setup]: CloseApplicationsFilter

Допустимые значения:
Список файловых шаблонов, разделённых запятыми

Значение по умолчанию:
*.exe,*.dll,*.chm

Описание

Эта директива выполняет ограничение записей секций [Files] и [InstallDelete], которые программа установки будет проверять на предмет использования. Будут проверены только те файлы, которые соответствуют одному из указанных шаблонов.

Установка значения *.* может обеспечить лучшую проверку за счёт скорости.

См. также:

CloseApplications
CloseApplicationsFilterExcludes
RestartApplications
