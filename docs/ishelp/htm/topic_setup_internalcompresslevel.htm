[Setup]: InternalCompressLevel

Допустимые значения:
none или одно из уровней сжатия LZMA

Значение по умолчанию:
normal

Описание

Определяет уровень сжатия LZMA, используемого во внутренних структурах программы установки. Как правило, нет никаких оснований для изменения значения по умолчанию normal.
