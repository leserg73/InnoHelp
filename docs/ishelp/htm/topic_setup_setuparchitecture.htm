[Setup]: SetupArchitecture

Допустимые значения:
x86, x64

Значение по умолчанию:
x86

Описание

При указании x86 (по умолчанию) используется 32-разрядная версия программы установки, которая работает на системах, способных запускать 32-разрядные двоичные файлы x86. К ним относятся системы под управлением x86 Windows, x64 Windows, а также Arm64 Windows 10 и 11, которые могут запускать двоичные файлы x86 посредством эмуляции.

В противном случае при указании x64 используется 64-разрядная версия программы установки, которая работает на системах, способных запускать двоичные файлы x64. К ним относятся системы под управлением x64 Windows, а также системы Windows 11 на базе Arm64, которые могут запускать двоичные файлы x64 посредством эмуляции.

Более подробную информацию см. в разделе Inno Setup 64-бит.
