Программирование: GetMD5OfUnicodeString

Прототип

function GetMD5OfUnicodeString(const S: String): String;

Описание

Возвращает строковое значение контрольной суммы MD5 для заданной строки в кодировке Unicode.

Пример

var
  MD5: String;
begin
  MD5 := GetMD5OfUnicodeString('Test');
  // MD5 = '8e06915d5f5d4f8754f51892d884c477'
end;
