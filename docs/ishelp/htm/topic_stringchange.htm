Препроцессор Inno Setup: StringChange

Прототип функции

str StringChange(str, str, str)

Описание

Возвращает копию строки (первый параметр), в которой искомое вхождение (второй параметр) заменено на другую строку (третий параметр).

Пример

; Объявляем переменную MyString2 и присваиваем ей значение 'a ca c'
#define MyString "a ca c"
; Выполняем в строке замену символов ' ' (пробел) на символ 'b'
#define MyString2 StringChange(MyString, " ", "b")
// Результат: переменная MyString2 = 'abcabc'
