Программирование: SaveStringToFile

Прототип

function SaveStringToFile(const FileName: String; const S: AnsiString; const Append: Boolean): Boolean;

Описание

Сохраняет указанную строку S в заданном файле FileName. Если параметр Append имеет значение True и заданный файл уже существует, то строки будут добавлены к файлу, вместо его полной перезаписи. В случае успеха возвращает значение True, иначе False.

Комментарии

Эта функция не выполняет автоматическую вставку переносов до или после строки. Если параметр Append имеет значение True и существующий файл завершается строкой без переноса, то функция фактически дополнит эту строку указанной. Чтобы избежать этого, можно поместить символы разрыва строки до и после строки так, как это показано в примере ниже:

SaveStringToFile('c:\filename.txt', #13#10 + 'the string' + #13#10, True);
