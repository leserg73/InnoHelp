Программирование: GetSysNativeDir

Прототип

function GetSysNativeDir: String;

Описание

Возвращает специальный псевдоним Sysnative без заключительной обратной косой черты. Если псевдонима Sysnative нет, возвращается пустая строка.

Псевдоним Sysnative недоступен для 64-разрядных установщиков и любых других 64-разрядных приложений, поэтому используйте его значение только для того, чтобы, например, передать его 32-разрядному приложению.
