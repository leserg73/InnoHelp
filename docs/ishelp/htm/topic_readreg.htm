Препроцессор Inno Setup: ReadReg

Прототип функции

any ReadReg(int RootKey, str SubKey, str? Name, any? Default)

Описание

Выполняет чтение значения из указанного ключа реестра Windows (32-бит или 64-бит). Первый параметр должен быть именем раздела реестра, например, HKEY_LOCAL_MACHINE, HKEY_LOCAL_MACHINE_32 или HKEY_LOCAL_MACHINE_64. Константы, используемые этим параметром, объявлены в файле ISPPBuiltins.iss. Второй параметр – это имя подраздела реестра. Третий параметр является ключом реестра, значение которого необходимо получить. Последний параметр (Default) является необязательным и может быть использован для установки значения, которое вернёт функция в случае неудачного выполнения операции.

Заметьте, функция может вернуть значение любого типа в зависимости от фактического типа данных в ключе реестра.
