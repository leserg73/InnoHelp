[Setup]: DirExistsWarning

Допустимые значения:
auto, yes или no

Значение по умолчанию:
auto

Описание

При значении auto программа установки выдаст сообщение "Папка ... уже существует. Всё равно установить в эту папку?", если на странице "Выбор папки установки" пользователь выберет существующий каталог, за исключением случаев, когда уже установлена другая версия такого же приложения и выбранный каталог совпадает с предыдущим (при условии, что директива UsePreviousAppDir имеет значение yes – используется по умолчанию).

При значении yes программа установки всегда будет показывать сообщение "Папка существует", когда пользователь выберет существующий каталог.

При значении no программа установки никогда не будет показывать сообщение "Папка существует".
