Программирование: IsIniSectionEmpty

Прототип

function IsIniSectionEmpty(const Section, Filename: String): Boolean;

Описание

Возвращает значение True, если указанный раздел в файле INI является пустым.

Параметр FileName определяет имя файла INI, в котором проверить следует раздел. Параметром Section задаётся имя раздела.
