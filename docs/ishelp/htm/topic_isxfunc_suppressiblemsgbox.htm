Программирование: SuppressibleMsgBox

Прототип

function SuppressibleMsgBox(const Text: String; const Typ: TMsgBoxType; const Buttons, Default: Integer): Integer;

Описание

Выводит на экран окно сообщения, которое может быть скрыто программой установки/удаления. Если окна сообщений блокируются (смотрите "Параметры командной строки программы установки"), то функция возвращает значение, заданное параметром Default. В противном случае поведение функции SuppressibleMsgBox аналогично функции MsgBox.
