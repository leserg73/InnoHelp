[Setup]: WizardBackColor

Допустимые значения:
Цвет в формате #rrggbb или $bbggrr, где bb, gg и rr определяют интенсивности (в шестнадцатеричном формате) для синего, зеленого и красного цветов соответственно. Или это может быть одно из следующих предопределённых названий цветов: black, maroon, green, olive, navy, purple, teal, gray, silver, red, lime, yellow, blue, fuchsia, aqua, white, none, window.

Значение по умолчанию:
(см. ниже)

Описание

Эта директива устанавливает цвет фона на страницах мастера установки и удаления, а также на всех других связанных окон, например, таких как диалог "Выбор языка".

Если используется фоновое изображение, указанное с помощью директивы WizardBackImageFile, то по умолчанию для этой директивы устанавливается значение window, а установка значения none не допускается.
В противном случае по умолчанию принимается значение none.

Использование значения window без указания фонового изображения не даёт никакого эффекта.

Установка значения none отключает эту функцию, поэтому будут использоваться родные цвета фона, определённые системой или стилем.

Если эта функция включена, а цвет установлен на значение, отличное от window, то все панели станут прозрачными. Чтобы отключить видимость всех разделителей (горизонтальные линии) в программе установки и удаления, добавьте в директиве WizardStyle модификатор hidebevels.

Кроме того, если эта функция включена, но в директиве WizardStyle не указан стиль, автоматически будет активирован стиль windows11 с модификатором excludelightcontrols.
