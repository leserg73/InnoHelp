[Setup]: CreateAppDir

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

Если задано значение no, папка приложения не будет создана, страница "Выбор папки установки" не будет показана и константа {app} будет эквивалентна константе {win}. Если при CreateAppDir=no включена функция создания деинсталлятора, данные деинсталляции будут созданы в системном каталоге Windows.
