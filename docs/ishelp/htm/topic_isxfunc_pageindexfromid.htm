Программирование: PageIndexFromID

Прототип

function PageIndexFromID(const ID: Integer): Integer;

Описание

По указанному ID страницы возвращает индекс её позиции. Используйте эту функцию, если, к примеру, хотите узнать, находится ли данная страница перед какой-нибудь другой.

Пример

function ShouldSkipPage(PageID: Integer): Boolean;
begin
  Result := PageIndexFromID(PageID) < PageIndexFromID(wpReady);
end;
