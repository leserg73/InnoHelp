Программирование: RemoveBackslash

Прототип

function RemoveBackslash(const S: String): String;

Описание

Возвращает указанную строку S без символа прямой/обратной косой черты в конце строки.

Комментарии

При обработке строк с путями файловой системы рекомендуется использовать функцию RemoveBackslashUnlessRoot вместо этой.
