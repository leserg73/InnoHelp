Программирование: GetIniString

Прототип

function GetIniString(const Section, Key, Default, Filename: String): String;

Описание

Считывает строковое значение (тип String) из файла INI.

Параметр FileName определяет имя файла INI, из которого следует получить значение ключа. Параметром Section задаётся имя раздела, а параметром Key — имя ключа.
