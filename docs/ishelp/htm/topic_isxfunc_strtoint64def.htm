Программирование: StrToInt64Def

Прототип

function StrToInt64Def(S: String; Def: Int64): Int64;

Описание

Преобразует числовую строку S в целочисленное 64-разрядное значение и типом Int64. Если результат преобразования строки S не содержит действительное 64-разрядное значение, функция вернёт 64-разрядное число, заданное в параметре Def.
