[Setup]: DefaultGroupName

Описание

Эта директива определяет имя папки по умолчанию в каталоге начального меню и используется на странице "Выбор папки в меню «Пуск»". Если директива не указана или является пустой, то для имени будет использоваться значение "(Default)".

Имейте в виду, что папки начального меню хранятся в виде литеральных каталогов, поэтому не разрешается указывать символы, запрещённые к использованию для обычных имён.

Пример:
DefaultGroupName=My Program
