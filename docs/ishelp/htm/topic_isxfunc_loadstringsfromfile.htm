Программирование: LoadStringsFromFile

Прототип

function LoadStringsFromFile(const FileName: String; var S: TArrayOfString): Boolean;

Описание

Загружает указанный текстовый файл FileName в массив строк S. В случае успеха возвращает значение True, иначе — False.

Комментарии

Используйте для загрузки функцию LoadStringFromLockedFile, что даст возможность прочитать файл, даже если он уже открыт для записи в другой программе.
