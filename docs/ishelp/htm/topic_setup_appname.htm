[Setup]: AppName

Описание

Эта директива является обязательной и определяет имя устанавливаемого приложения. Здесь не следует указывать номер версии приложения, т.к. для него предназначены другие директивы: AppVersion и/или AppVerName. Значение AppName используется в заголовке окна программы установки и удаления, на страницах мастера, а также в сообщениях и диалогах. Может содержать константы.

Если директиве DisableWelcomePage задано значение yes (является значением по умолчанию), то в заголовках окон используется значение директивы AppVerName вместо AppName.

Значение этой директивы также используется в качестве значения по умолчанию для директив AppId, VersionInfoDescription и VersionInfoProductName, если они не указаны.

Пример:
AppName=My Program
