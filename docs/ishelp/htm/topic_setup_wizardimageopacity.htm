[Setup]: WizardImageOpacity

Допустимые значения:
от 0 до 255

Значение по умолчанию:
255

Описание

Эта директива устанавливает непрозрачность изображений мастера. При указании значения 0 изображения мастера будут полностью прозрачными. При указании значения 255 – полностью непрозрачными.

Эта функция может пригодиться, если, к примеру, необходимо использовать одни и те же изображения как для обычного стиля, так и для тёмного, с директивами DynamicDark. Тогда при установке значения 150 можно добиться частичной прозрачности изображений, что они одинаково гармонировали как со светлой темой, так и с тёмной.

Пример:
WizardStyle=modern dynamic
WizardImageFile=MyImage.png
WizardImageFileDynamicDark=MyImage.png
WizardSmallImageFile=MySmallImage.png
WizardSmallImageFileDynamicDark=MySmallImage.png
WizardImageOpacity=150
