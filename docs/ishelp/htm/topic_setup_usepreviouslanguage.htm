[Setup]: UsePreviousLanguage

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

При значении yes программа установки во время своего запуска выполнит в реестре поиск информации об установке того же приложения и, если таковая будет найдена, использует язык предыдущей установки в качестве языка по умолчанию для текущей установки.

Заметьте, что этот параметр не изменяет язык, используемый в самом диалоге "Выбор языка". Более подробную информацию о том, какой язык используется по умолчанию в диалоге "Выбор языка", смотрите в разделе справки Секция [Languages].

Также обратите внимание, что программа установки не может повторно использовать настройки предыдущей установки, у которой директива Uninstallable имела значение no, поскольку искомые записи реестра не создаются, если Uninstallable=no.

Директива UsePreviousLanguage должна иметь значение no, если параметр AppId содержит константы.
