[Setup]: DisablePrecompiledFileVerifications

Допустимые значения:
Перечень следующих значений, разделённых пробелами:
setup
setupcustomstyle
setupldr
is7z
isbunzip
isunzlib
islzma

Значение по умолчанию:
пустое

Описание

Перед использованием предварительно скомпилированных файлов, компилятор сначала проверяет их подлинность. Для некоторых из них эту проверку, при необходимости, можно отключить. Используйте эту директиву для указания, файлов, которые не должны проверяться.

Значение setupldr применяется как для setupldr.e32, так и для setupldr.e64. Значение is7z применяется не только к файлу isz7.dll, но также к is7zxr.dll и is7zxa.dll. Аналогично значение islzmaexe применяется как для islzma32.exe, так и для islzma64.exe.
