Программирование: ComparePackedVersion

Прототип

function ComparePackedVersion(const Version1, Version2: Int64): Integer;

Описание

Выполняет сравнение упакованных значений версий Version1 с Version2.

  Возвращает одно из следующих значений:
   < 0 — если Version1 меньше Version2;
      0 — если Version1 равно Version2;
   > 0 — если Version1 больше Version2.
