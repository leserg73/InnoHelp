Pascal Scripting: ExpandFileName

Прототип

function ExpandFileName(const FileName: string): String;

Описание

Возвращает полный путь для указанного имени файла FileName. Полностью определённый путь включает букву диска, имени каталога и подкаталога, в котором находится файл, а также имя файла и его расширение.
