Программирование: PathConvertNormalToSuper

Прототип

function PathConvertNormalToSuper(const Filename: String): Boolean;

Описание

Выполняет преобразование обычного пути в полный удлинённый путь (также известный как "суперпуть"). Никакой ошибки не будет, если указанный путь уже является удлиненным путём.

Если входные данные представляют собой пустую строку или содержат только пробелы, возвращается пустая строка.

В случае сбоя генерируется исключение; известно, что это происходит только в том случае, если длина входного или выходного пути превышает 32 КБ.

Пример

var
  SuperPath: String;
begin
  SuperPath := PathConvertNormalToSuper('C:\Windows') then
  Log(SuperPath);  // Logs '\\?\C:\Windows'

  SuperPath := PathConvertNormalToSuper('\\server\share\filename') then
  Log(SuperPath);  // Logs '\\?\UNC\server\share\filename'
end;
