Программирование: SelectDisk

Прототип

function SelectDisk(const DiskNumber: Integer; const AFilename: String; var Path: String): Boolean;

Описание

Выводит на экран сообщение "Необходимо вставить следующий диск" и возвращает значение True, если пользователь вводит путь, который содержит файл, указанный в параметре AFilename, и нажимает "OK". Выбранный путь возвращается в параметре Path. Текущее значение параметра Path используется в качестве первоначально выбранного пути, а значение параметра DiskNumber служит только для информирования пользователя.
