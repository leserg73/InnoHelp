[Setup]: CreateUninstallRegKey

Допустимые значения:
yes или no, или логическое выражение в сценарии

Значение по умолчанию:
yes

Описание

Если задано значение no или логическое выражение в сценарии вернёт значение False, программа установки не будет создавать запись в апплете Панели управления "Удаление или изменение программы".

Установка для этой директивы значения no может использоваться при создании обновления для существующего приложения, когда не нужно создавать другую запись деинсталляции, но при этом необходимо сохранить функции удаления не отключая их полностью (например, с помощью директивы Uninstallable=no). Иначе, для директивы UpdateUninstallLogAppName, как правило, также устанавливается значение no.
