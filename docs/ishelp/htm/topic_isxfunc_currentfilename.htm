Программирование: CurrentFilename

Прототип

function CurrentFilename: String;

Описание

Возвращает имя целевого файла секции [Files], обрабатываемого в текущий момент. Полученное имя файла может содержать константы.

Не пытайтесь вызывать эту функцию за пределами функций Check, BeforeInstall или AfterInstall, принадлежащих секции [Files].
