Параметры командной строки деинсталлятора

Программа удаления (unins???.exe) имеет необязательные параметры командной строки. Они могут быть полезны системным администраторам для вызова программы удаления из других приложений.

Смотрите также раздел "Параметры командной строки инсталлятора".

/SILENT, /VERYSILENT

Если задан этот параметр, программа удаления не будет запрашивать у пользователя подтверждения об удалении или показывать сообщение о том, что удаление завершено. Общие файлы, которые больше не используются, удаляются автоматически без запроса. Любые критические сообщения об ошибках по-прежнему будут отображаться на экране. Если же задан параметр "/VERYSILENT", то не будет показано окно с процессом удаления.

Если требуется перезагрузка, а параметр "/NORESTART" не используется и указан "/VERYSILENT", деинсталлятор выполнит перезагрузку без предупреждения.

/SUPPRESSMSGBOXES

Указывает деинсталлятору подавлять окна сообщений. Команда действительна только в сочетании с параметрами "/SILENT" и "/VERYSILENT". Подробное описание "/SUPPRESSMSGBOXES" см. в разделе Параметры командной строки инсталлятора.

/LOG

Указывает деинсталлятору создать в пользовательском каталоге TEMP файл отчёта, в который записываются события удаления файлов и действия секции [UninstallRun], предпринятые в процессе деинсталляции. Этот файл может быть полезен для отладки работы деинсталлятора.

При создании файл отчёта получает уникальное имя на основе текущей даты, таким образом существующие файлы не будут дополнены или перезаписаны.

Информация, содержащаяся в файле отчёта, носит технический характер и адресуется в первую очередь опытным пользователям и специалистам. Формат файла не предназначен для машинного анализа и может быть изменён без предварительного уведомления.

/LOG="filename"

Этот параметр аналогичен "/LOG", но в отличие от него позволяет задать определённый путь/имя файла для записи отчёта деинсталляции. Если файл с указанным именем уже существует, он будет перезаписан. Если файл создать не удастся, деинсталляция будет прервана с сообщением об ошибке.

/NORESTART

Указывает деинсталлятору не перезагружать систему, даже если это необходимо.
