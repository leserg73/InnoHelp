Программирование: ExtractFilePath

Прототип

function ExtractFilePath(const FileName: String): String;

Описание

Извлекает диск и каталог для указанного имени файла, включая завершающий символ обратной косой черты. Итоговая строка будет содержать все символы левой части FileName, включая двоеточие или обратную косую черту, которые отделяют имя пути от имени файла с расширением. Если заданное имя не содержит определение пути (диск и каталог), то результатом будет пустая строка.
