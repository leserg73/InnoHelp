[Setup]: WizardStyle

Допустимые значения:
classic или modern

Значение по умолчанию:
classic

Описание

При указании этой директивы со значением modern внешний вид программы установки и удаления примет более современный вид. Отдельно для программы установки также изменятся значения по умолчанию у директив WizardResizable и WizardSizePercent на yes и 120,120 соответственно.

Чтобы добавить полную поддержку WizardResizable и WizardSizePercent для всех пользовательских элементов управления и страниц мастера, если таковые имеются, используйте свойства Anchors и KeepSizeY. В качестве примера смотрите файл сценария CodeClasses.iss.
