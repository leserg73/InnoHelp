Программирование: EnableFsRedirection

Прототип

function EnableFsRedirection(const Enable: Boolean): Boolean;

Описание

Функция удалена в версии 7.0.

Комментарии

Вместо этой функции используйте ApplyPathRedirRulesForCurrentProcess или 64-разрядный установщик.

В 32-разрядных установщиках вместо этой функции также можно использовать константу {sysnative}.

Для запуска исполняемого файла, расположенного в 64-разрядном системном каталоге, из 32-разрядного установщика используйте вспомогательную функцию ExecWithNativeSysDir, ExecAndLogOutputWithNativeSysDir или ExecAndCaptureOutputWithNativeSysDir.
