Программирование: PathCombine

Прототип

function PathCombine(const Dir, Filename: String): String;

Описание

Объединяет каталог и имя файла в полный путь. Если Filename пуст, результатом будет пустая строка. Если Filename уже начинается с буквы диска или обратной косой черты (обратный слеш), он возвращается без изменений. В противном случае, при необходимости, между Dir и Filename вставляется один обратный слэш.
