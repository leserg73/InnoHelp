Программирование: CreateOutputMarqueeProgressPage

Прототип

function CreateOutputMarqueeProgressPage(const ACaption, ADescription: String): TOutputMarqueeProgressWizardPage;

Описание

Выполняет создание страницы мастера установки, которая содержит статичный текст и непрерывный (стиль Marquee) индикатор выполнения.

Параметр ACaption определяет текст для заголовка страницы, параметр ADescription — краткое описание назначения страницы.

Информацию о создании страниц с индикатором выполнения смотрите в разделе CreateOutputProgressPage.

Комментарии

Для анимации индикатора выполнения используйте вызов метода Animate. Не используйте вызов метода SetProgress, это приведёт к внутренней ошибке.

Пример

Смотрите файл AllPagesExample.iss из папки "Examples" в каталоге установки Inno Setup.
