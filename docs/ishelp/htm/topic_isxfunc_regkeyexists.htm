Программирование: RegKeyExists

Прототип

function RegKeyExists(const RootKey: Integer; const SubKeyName: String): Boolean;

Описание

Возвращает значение True, если указанный ключ реестра существуют.

Пример

begin
  if RegKeyExists(HKEY_AUTO, 'Software\Jordan Russell\Inno Setup') then
  begin
    // Такой ключ существует
  end;
end;
