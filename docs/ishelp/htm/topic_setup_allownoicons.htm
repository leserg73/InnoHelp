[Setup]: AllowNoIcons

Допустимые значения:
yes или no

Значение по умолчанию:
no

Описание

Если директива имеет значение yes, на странице "Выбор папки в меню «Пуск»" программа установки отобразит флажок "Не создавать папку в меню «Пуск»", что позволит пользователю пропустить создание ярлыков в меню «Пуск».

По умолчанию влияет только на те записи секции [Icons], в которых параметр Name начинается с {group}\ и не содержит параметр Tasks. Для воздействия на другие записи секции [Icons], необходимо добавить параметр Check: not WizardNoIcons.
