Pascal Scripting: SetIniBool

Прототип

function SetIniBool(const Section, Key: String; const Value: Boolean; const Filename: String): Boolean;

Описание

Выполняет запись логического значения (тип Boolean) в файл INI.

Параметр FileName определяет имя файла INI. Параметром Section задаётся имя раздела, а параметром Key — имя ключа. В параметре Value указывается желаемое значение (True или False).
