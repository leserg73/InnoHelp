Препроцессор Inno Setup: FindSectionEnd

Прототип функции

int FindSectionEnd(str Section = "Files")

Описание

Возвращает индекс (начиная с нуля) строки, которая является последней записью для искомой секции. Если секция не найдена, возвращает значение меньше нуля Предназначена для использования с директивой insert.

Функция объявлена в файле ISPPBuiltins.iss.
