Препроцессор Inno Setup: EmitLanguagesSection

Прототип функции

void EmitLanguagesSection

Описание

Заполняет секцию [Languages] всеми доступными официальными языковыми файлами, включая английский.

После компиляции программы установки откройте вкладку "Вывод препроцессора" в среде разработки IDE компилятора, чтобы увидеть полный список доступных языков, включая их автоматически сгенерированные имена.

Функция объявлена в файле ISPPBuiltins.iss.

Пример

#expr EmitLanguagesSection
