Программирование: ExpandConstant

Прототип

function ExpandConstant(const S: String): String;

Описание

Заменяет все константы в строке S на соответствующие им значения. Например, ExpandConstant('{srcexe}') будет изменено на имя файла программы установки.

Если при расширении константы произойдет ошибка, будет вызвано исключение.
