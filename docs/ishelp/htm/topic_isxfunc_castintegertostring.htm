Программирование: CastIntegerToString

Прототип

function CastIntegerToString(const L: Longint): String;

Описание

Выполняет приведение целого числа в строку, что позволяет получать строковые данные из файла DLL посредством CallDllProc.

Комментарии

Эта функция устарела. Смотрите раздел справки Использование DLL.
