[Setup]: DiskSliceSize

Допустимые значения:
от 262144 до 9223372036800000000, или max

Значение по умолчанию:
2100000000

Описание

Определяет максимальное количество байт на сегмент диска (файл Setup-*.bin). Обычно это значение должно быть равно общему количеству байтов, доступных на дисковом носителе, делённому на значение директивы SlicesPerDisk секции [Setup], которая по умолчанию имеет значение 1.

Эта директива не учитывается, если не включено разделение данных на диски при помощи DiskSpanning в секции [Setup].

Чтобы оптимально заполнить DVD диск размером 4,7 Гб, используйте:

SlicesPerDisk=3
DiskSliceSize=1566000000

Чтобы оптимально заполнить DVD диск размером 8,5 Гб (двухслойный), используйте:

SlicesPerDisk=5
DiskSliceSize=1708200000

Чтобы оптимально заполнить CD диск размером 700 Мб (80-минут), используйте:

SlicesPerDisk=1
DiskSliceSize=736000000
