Программирование: IsX86OS

Прототип

function IsX86OS: Boolean;

Описание

Возвращает значение True, если система соответствует идентификатору архитектуры x86os.

До версии Inno Setup 6.3 функция IsX86OS называлась IsX86. Для совместимости с предыдущими версиями Inno компилятор по прежнему принимает IsX86 в качестве псевдонима для IsX86OS.
