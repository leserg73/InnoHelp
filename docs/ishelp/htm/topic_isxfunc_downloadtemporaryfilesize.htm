Программирование: DownloadTemporaryFileSize

Прототип

function DownloadTemporaryFileSize(const Url): Int64;

Описание

Возвращает размер файла по указанному URL-адресу без загрузки самого файла. Если сервер не предоставляет такой информации, функция вернёт значение -1.

В случае ошибки будет вызвано исключение.

Дополнительные сведения смотрите в описании функции DownloadTemporaryFile.
