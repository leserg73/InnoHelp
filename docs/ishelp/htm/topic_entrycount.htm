Препроцессор Inno Setup: EntryCount

Прототип функции

int EntryCount(str)

Описание

Возвращает суммарное количество действительных строк в указанной секции текущего вывода препроцессора. Пустые строки и комментария НЕ учитываются. Функция обрабатывает несколько секций с одинаковыми именами и подсчитывает все из них.
