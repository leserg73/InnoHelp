Препроцессор Inno Setup: GetSHA256OfString

Прототип функции

str GetSHA256OfString(str)

Описание

Возвращает строковое значение контрольной суммы SHA-256 для заданной строки в кодировке ANSI.

Пример

// Объявляем переменную SHA256 и присваиваем ей значение
// контрольной суммы строки "Test", которое вернёт функция.
#define SHA256 GetSHA256OfString('Test')
// SHA256 = '532eaabd9574880dbf76b9b8cc00832c20a6ec113d682299550d7a6e0f345e25'
