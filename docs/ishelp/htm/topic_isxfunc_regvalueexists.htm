Программирование: RegValueExists

Прототип

function RegValueExists(const RootKey: Integer; const SubKeyName, ValueName: String): Boolean;

Описание

Возвращает значение True, если указанный ключ реестра и значение существуют.

Пример

begin
  if RegValueExists(HKEY_CURRENT_USER, 'Console', 'WindowSize') then
  begin
    // Такое значение существует
  end;
end;
