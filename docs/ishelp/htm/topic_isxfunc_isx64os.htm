Программирование: IsX64OS

Прототип

function IsX64OS: Boolean;

Описание

Возвращает значение True, если система соответствует идентификатору архитектуры x64os.

До версии Inno Setup 6.3 функция IsX64OS называлась IsX64. Для совместимости с предыдущими версиями Inno компилятор по прежнему принимает IsX64 в качестве псевдонима для IsX64OS.
