[Setup]: ReserveBytes

Значение по умолчанию:
0

Описание

Определяет минимальный размер пустого места (в байтах), которое будет зарезервировано на первом диске. Эта функция может пригодиться, когда помимо файлов программы установки на диск нужно будет скопировать другие файлы (например, ReadMe и др.).

Компилятор выполнит округление заданного числа до размеров ближайшего кластера.

Эта директива не учитывается, если не включено разделение данных на диски при помощи DiskSpanning в секции [Setup].
