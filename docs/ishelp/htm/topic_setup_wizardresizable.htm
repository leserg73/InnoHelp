[Setup]: WizardResizable

Допустимые значения:
yes или no

Значение по умолчанию:
yes, если WizardStyle=modern, иначе no

Описание

При установке значения yes пользователь сможет изменить размер главного окна мастера установки.

Чтобы добавить полную поддержку WizardResizable и WizardSizePercent для всех пользовательских элементов управления, страниц мастера и форм TSetupForm, если таковые имеются, используйте свойства Anchors и KeepSizeY. В качестве примера смотрите файл сценария CodeClasses.iss.
