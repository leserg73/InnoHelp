Препроцессор Inno Setup: FindCode

Прототип функции

int FindCode()

Описание

Возвращает индекс (начиная с нуля) строки, следующей за строкой объявления секции [Code] или ключевым словом "program". Возвращает значение меньше нуля, если ни то, ни другое не найдено.

Функция объявлена в файле ISPPBuiltins.iss.
