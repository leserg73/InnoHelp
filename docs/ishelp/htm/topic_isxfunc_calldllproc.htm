Pascal Scripting: CallDLLProc

Прототип

function CallDLLProc(const DLLHandle: Longint; const ProcName: String; const Param1, Param2: Longint; var Result: Longint): Boolean;

Описание

Выполняет вызов указанной функции из библиотеки DLL, определённой с помощью дескриптора DLL, который вернула функция LoadDLL. Возвращает значение True, если вызов процедуры был успешный, в противном случае False.

Функция должна использовать стандартное соглашение о вызовах, принимать два 4-байтных целочисленных параметра и возвращать 4-байтный целочисленный результат.

Комментарии

Эта функция устарела. Смотрите раздел справки Использование DLL.
