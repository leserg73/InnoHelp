[Setup]: DisableProgramGroupPage

Допустимые значения:
auto, yes или no

Значение по умолчанию:
auto

Описание

Эта директива управляет отображением страницы с выбором папки в меню «Пуск».

Если задано значение yes, программа установки не будет показывать страницу "Выбор папки в меню «Пуск»".

Если директива имеет значение auto, тогда при запуске программа установки выполнит в реестре поиск идентификатора такого же приложения и в случае его обнаружения страница "Выбор папки в меню «Пуск»" показана не будет.

Если страница мастера "Выбор папки в меню «Пуск»" не отображается, для меню «Пуск» всегда используется имя папки по умолчанию.

См. также AlwaysShowGroupOnReadyPage.
