Программирование: PackVersionNumbers

Прототип

function PackVersionNumbers(const VersionMS, VersionLS: Cardinal): Int64;

Описание

Выполняет упаковку указанных номеров версии в одно значение.

Для сравнения упакованных версий всегда используйте функцию ComparePackedVersion или SamePackedVersion.
