Программирование: BrowseForFolder

Прототип

function BrowseForFolder(const Prompt: String; var Directory: String; const NewFolderButton: Boolean): Boolean;

Описание

Отображает диалог, который позволяет пользователю выбрать желаемую папку. Первоначально выбранная папка определяется значением параметра Directory. Если для параметра NewFolderButton указано значение True, то в диалоге будет показана кнопка "Создать папку", с помощью которой пользователь сможет создать новую папку. Возвращает значение True, если пользователь папку выбрал и нажал "OK", в противном случае False. Выбранная папка возвращается параметром Directory. В параметре Promt указывается пояснение для пользователя, например, "Выберите папку из списка и нажмите «ОК»", которое будет показано в диалоге.
