[Setup]: CloseApplicationsFilterExcludes

Допустимые значения:
Список файловых шаблонов, разделённых запятыми

Описание

Эта директива выполняет ограничение записей секций [Files] и [InstallDelete], которые программа установки будет проверять на предмет использования. Файлы, которые соответствуют одному из указанных шаблонов, не будут проверяться, даже если они удовлетворяют условиям CloseApplicationsFilter.

Например, укажем проверку всех файлов за исключением файла MyProg.exe:

Пример:
[Setup]
CloseApplicationsFilter=*.*
CloseApplicationsFilterExcludes=MyProg.exe

См. также:

CloseApplications
CloseApplicationsFilter
RestartApplications
