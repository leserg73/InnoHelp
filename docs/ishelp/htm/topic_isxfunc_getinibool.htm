Программирование: GetIniBool

Прототип

function GetIniBool(const Section, Key: String; const Default: Boolean; const Filename: String): Boolean

Описание

Считывает логическое значение (тип Boolean) из файла INI.

Параметр FileName определяет имя файла INI, из которого следует получить значение ключа. Параметром Section задаётся имя раздела, а параметром Key — имя ключа. С помощью параметра Default указывается логическое значение по умолчанию, которое вернёт функция в случае непредвиденных ошибок.
