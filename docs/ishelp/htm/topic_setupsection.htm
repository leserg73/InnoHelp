Секция [Setup]


Эта секция содержит глобальные директивы, которые используются как программой установки, так и программой удаления. Определённые директивы этой секции необходимы для создания любого инсталлятора. Ниже приведен пример секции [Setup]:
[Setup]
AppName=My Program
AppVersion=1.5
DefaultDirName={autopf}\My Program
DefaultGroupName=My Program
По умолчанию все начальные и конечные пробелы в значении директивы будут удалены. Этого можно избежать, если заключить значение параметра в двойные кавычки (").
