Препроцессор Inno Setup: GetFileDateTimeString

Прототип функции

str GetFileDateTimeString(str Filename, str DateTimeFormat, str? DateSeparator, str? TimeSeparator)

Описание

Возвращает строковое значение даты и времени для заданного файла.

Первый параметр является именем файла (вместе с расширением). Второй параметр определяет формат даты и времени. Третий и четвертый параметры являются необязательными и устанавливают символ разделителя для компонентов даты (DateSeparator) и времени (TimeSeparator) соответственно. Спецификаторы формата даты и времени подробно изложены в разделе GetDateTimeString.

Пример

// Объявляем переменную MyFileDatetimeString и присваиваем ей строковое значение
// даты и времени файла myfile.txt, которое вернёт функция GetFileDateTimeString.
#define MyFileDateTimeString GetFileDateTimeString('myfile.txt', 'dd/mm/yyyy hh:nn:ss', '-', ':');
