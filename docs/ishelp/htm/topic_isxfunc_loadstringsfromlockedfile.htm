Программирование: LoadStringsFromLockedFile

Прототип

function LoadStringsFromLockedFile(const FileName: String; var S: TArrayOfString): Boolean;

Описание

Загружает указанный текстовый файл FileName в массив строк S, даже если файл уже открыт для записи в другой программе. Также поддерживаются файлы в кодировке UTF-8 с или без маркера BOM. В случае успеха функция возвращает значение True, иначе — False.
