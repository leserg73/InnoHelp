[Setup]: VersionInfoVersion

Допустимые значения:
Значение в виде 4-х чисел, разделённых точкой

Значение по умолчанию:
0.0.0.0

Описание

Определяет значение версии двоичного файла для сведений о версии программы установки.

Допускается указывать частичные номера версий. Пропущенные числа будут добавлены в виде нулей.

Эта директива не учитывается, если UseSetupLdr имеет значение no.
