[Setup]: WizardImageBackColor

Допустимые значения:
Цвет в формате #rrggbb или $bbggrr, где bb, gg и rr определяют интенсивности (в шестнадцатеричном формате) для синего, зеленого и красного цветов соответственно. Или это может быть одно из следующих предопределённых названий цветов: clBlack, clMaroon, clGreen, clOlive, clNavy, clPurple, clTeal, clGray, clSilver, clRed, clLime, clYellow, clBlue, clFuchsia, clAqua, clWhite, clNone, clWindow.

Значение по умолчанию:
(см. ниже)

Описание

Эта директива устанавливает цвет фона, используемого для заполнения всех неиспользуемых или прозрачных областей в изображении мастера установки, которое задаётся директивой WizardImageFile. Использование значения clNone в качестве цвета заливки сохраняет прозрачность.

Если используется пользовательское изображение, то по умолчанию эта директива принимает значение clWindow (обычно белый).

Если используется предустановленный стиль slate, то по умолчанию для этой директивы устанавливается значение #bcd2e2 (светло-серовато-синий).

Если используется предустановленный стиль zircon, то по умолчанию устанавливается значение #d0eaee (светло-серовато-голубой).

Если принудительно используется тёмный стиль – по умолчанию #2e3a3f (очень тёмно-синий).

В противном случае – #e8f3f9 (светло-серовато-синий).
