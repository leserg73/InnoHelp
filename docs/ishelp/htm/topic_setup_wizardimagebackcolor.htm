[Setup]: WizardImageBackColor

Допустимые значения:
Цвет в формате $bbggrr, где bb, gg и rr определяют интенсивности (в шестнадцатеричном формате) для синего, зеленого и красного цветов соответственно. Или это может быть одно из следующих предопределённых названий цветов: clBlack, clMaroon, clGreen, clOlive, clNavy, clPurple, clTeal, clGray, clSilver, clRed, clLime, clYellow, clBlue, clFuchsia, clAqua, clWhite, clNone, clWindow.

Значение по умолчанию:
$f9f3e8 (синевато-серый), если используется встроенное изображение мастера, в противном случае — clWindow.

Описание

Эта директива устанавливает цвет фона, используемого для заполнения всех неиспользуемых или прозрачных областей в изображении мастера установки, которое задаётся директивой WizardImageFile.
