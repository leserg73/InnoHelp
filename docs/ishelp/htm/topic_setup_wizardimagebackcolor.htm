[Setup]: WizardImageBackColor

Допустимые значения:
Цвет в формате #rrggbb или $bbggrr, где bb, gg и rr определяют интенсивности (в шестнадцатеричном формате) для синего, зеленого и красного цветов соответственно. Или это может быть одно из следующих предопределённых названий цветов: black, maroon, green, olive, navy, purple, teal, gray, silver, red, lime, yellow, blue, fuchsia, aqua, white, none, window.

Значение по умолчанию:
(см. ниже)

Описание

Эта директива устанавливает цвет фона, используемого для заполнения всех неиспользуемых или прозрачных областей в изображении мастера установки, которое задаётся директивой WizardImageFile. Использование значения none в качестве цвета заливки сохраняет прозрачность.

Если используется фоновое изображение, указанное с помощью директивы WizardBackImageFile, то по умолчанию для этой директивы устанавливается значение none.

Если же для WizardImageFile указано пустое значение, то по умолчанию также принимается значение none.

Если используется пользовательское изображение мастера, то по умолчанию эта директива принимает значение window (обычно белый).

Если используется предустановленный стиль slate, то по умолчанию для этой директивы устанавливается значение #bcd2e2 (светло-серовато-синий).

Если используется предустановленный стиль zircon, то по умолчанию устанавливается значение #d0eaee (светло-серовато-голубой).

Если принудительно используется тёмный стиль – по умолчанию #2e3a3f (очень тёмно-синий).

В противном случае – #e8f3f9 (светло-серовато-синий).
