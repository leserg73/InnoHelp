Программирование: RegisterExtraCloseApplicationsResource

Прототип

function RegisterExtraCloseApplicationsResource(const DisableFsRedir: Boolean; const AFilename: String): Boolean;

Описание

Выполняет регистрацию дополнительного файла, который программа установки должна проверить на предмет использования. Вызов функции RegisterExtraCloseApplicationsResource необходимо выполнять внутри события RegisterExtraCloseApplicationsResources для каждого файла. При этом фильтр CloseApplicationsFilter не учитывается. В случае успеха функция возвращает значение True.
