Препроцессор Inno Setup: LowerCase

Прототип функции

str LowerCase(str)

Описание

Возвращает строку с тем же самым текстом, в котором все буквы преобразованы к нижнему регистру (строчные). Функция LowerCase выполняет преобразование только для символов в пределах "A" и "Z" из 7-бит таблицы ASCII.

Примечание: операторы сравнения строк не чувствительны к регистру, поэтому перед сравнением не нужно применять к строкам эту функцию.
