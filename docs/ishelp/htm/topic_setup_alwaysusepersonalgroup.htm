[Setup]: AlwaysUsePersonalGroup

Допустимые значения:
yes или no

Значение по умолчанию:
no

Описание

Обычно константа {group} указывает на общую для всех пользователей папку начального меню, если текущий пользователь имеет права администратора. Если эта директива имеет значение yes, то вместо общей папки всегда будет использоваться папка текущего пользователя.

При использовании этой директивы следует проявлять осторожность, т.к. результат может оказаться абсолютно противоположный от ожидаемого. Компилятор будет предупреждать вас об этом, хотя это предупреждение можно отключить с помощью UsedUserAreasWarning.
