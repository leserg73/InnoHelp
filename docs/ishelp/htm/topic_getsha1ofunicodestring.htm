Препроцессор Inno Setup: GetSHA1OfUnicodeString

Прототип функции

str GetSHA1OfUnicodeString(str)

Описание

Возвращает строковое значение контрольной суммы SHA-1 для заданной строки в кодировке Unicode.

Пример

// Объявляем переменную SHA1 и присваиваем ей значение
// контрольной суммы строки "Test", которое вернёт функция.
#define SHA1 GetSHA1OfUnicodeString('Test')
// SHA1 = '9ab696a37604d665dc97134dbee44cfe70451b1a'
