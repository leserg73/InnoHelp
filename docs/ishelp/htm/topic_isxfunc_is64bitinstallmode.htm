Программирование: Is64BitInstallMode

Прототип

function Is64BitInstallMode: Boolean;

Описание

Возвращает значение True, если программа установки/удаления запущена в режиме 64-bit. Если запуск выполнен в режиме 32-bit, то функция вернёт значение False.

Комментарии

Если функция вернула True, то можно с уверенностью предположить, что функция IsWin64 также вернёт значение True.

Пример

begin
  if Is64BitInstallMode then
    MsgBox('Установка в режиме 64-бит', mbInformation, MB_OK)
  else
    MsgBox('Установка в режиме 32-бит', mbInformation, MB_OK);
end;
