Программирование: GetSHA1OfString

Прототип

function GetSHA1OfString(const S: AnsiString): String;

Описание

Возвращает строковое значение контрольной суммы SHA-1 для заданной строки в кодировке ANSI.

Пример

var
  SHA1: String;
begin
  SHA1 := GetSHA1OfString('Test');
  // SHA1 = '640ab2bae07bedc4c163f679a746f7ab7fb5d1fa'
end;
