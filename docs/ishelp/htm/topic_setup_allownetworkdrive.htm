[Setup]: AllowNetworkDrive

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

Если задано значение no, то пользователю не будет разрешено ввести сетевой путь на странице "Выбор папки установки".

Чтобы полностью запретить установку в сетевые папки, следует также для директивы AllowUNCPath указать значение no.
