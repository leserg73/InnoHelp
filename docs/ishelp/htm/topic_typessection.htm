Секция [Types]

Эта секция является необязательной. Она позволяет задать все типы установок, которые будут показаны программой установки на странице "Выбор компонентов". Если типы не указаны, но определены в секции [Components], то во время компиляции будет создан набор типов с настройками по умолчанию. Если используется файл сообщений по умолчанию (English), то описания типов установки будут такими же, как показаны в примере ниже.

Пример секции [Types]:

[Types]
Name: "full"; Description: "Полная установка"
Name: "compact"; Description: "Компактная установка"
Name: "custom"; Description: "Выборочная установка"; Flags: iscustom

Ниже приведен список поддерживаемых параметров:

Name  (обязательный)

Определяет внутреннее имя типа установки. Используется в качестве параметра для компонентов в секции [Components] и указывает программе установки, к каким типам относится компонент.

Пример:
Name: "full"
Description  (обязательный)

Описание типа установки (может содержать константы). Это описание отображается во время установки.

Пример:
Description: "Полная установка"
Flags

Этот параметр представляет собой набор дополнительных опций. Допускается использовать несколько опций, разделяя их пробелами. Поддерживаются следующие опции:

iscustom

Указывает программе установки, что данный тип является выборочным. Всякий раз, когда пользователь вручную изменяет набор устанавливаемых компонентов, программа установки изменяет тип установки на выборочный. Обратите внимание, если выборочный тип не будет задан, то программа установки позволит пользователю выбрать только тип установки, а изменить состав компонентов вручную пользователь уже не сможет.

Этот флаг может содержать только один тип установки.

Пример:
Flags: iscustom
