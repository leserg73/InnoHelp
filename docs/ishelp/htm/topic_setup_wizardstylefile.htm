Описание

Эта директива определяет пользовательский файл VCL стиля (.vsf). Рекомендуется использовать стили только с поддержкой высокого DPI.

При использовании WizardStyle с параметром light или dynamic указанный стиль должен использовать светлую цветовую схему (например, преимущественно белый или другие светлые цвета). При использовании параметра dark стиль должен быть с тёмной цветовой схемой (например, преимущественно чёрный или другие тёмные цвета).

Применение пользовательского файла стиля увеличивает размер установщика, за исключением случаев, когда используется режим dark и файл стиля, размер которого меньше встроенного и составляет не более 199 КБ до сжатия.

Также вместо файла можно указать builtin:polar, builtin:slate или builtin:zircon, чтобы использовать один из предустановленных стилей. Дополнительную информацию смотрите в разделе WizardStyle.

Файлы VCL стилей можно найти и загрузить и таких источников, как GetIt (бесплатно, если в поле "Subscription" указано "Not Required"; для загрузки требуется также установка среды разработчика Delphi, например, в редакции "Community Edition" для бесплатных стилей) и DelphiStyles (не бесплатно).

Файлы стилей FireMonkey (FMX) не поддерживаются.
