Программирование: InitializeBitmapButtonFromIcon

Прототип

function InitializeBitmapButtonFromIcon(const BitmapButton: TBitmapButton; const IconFilename: String; const BkColor: TColor; const AscendingTrySizes: TArrayOfInteger): Boolean;

Описание

Выполняет инициализацию на заданной кнопке растрового изображения файла значка, указанного в параметре IconFilename, используя заданный цвет фона BkColor для прозрачных частей изображения.

Для получения дополнительной информации смотрите описание функции InitializeBitmapImageFromIcon.
