Программирование: ExpandUNCFileName

Прототип

function ExpandUNCFileName(const FileName: String): String;

Описание

Возвращает полный путь для указанного имени файла FileName. Полностью определённый путь включает имя диска в формате UNC '\\<servername>\<sharename>', если буква диска сопоставляется с сетевым ресурсом вместо локального диска, имени каталога и подкаталога, в котором находится файл, а также имени файла с расширением.
