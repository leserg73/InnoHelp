[Setup]: WizardBackImageOpacity

Допустимые значения:
от 0 до 255

Значение по умолчанию:
255

Описание

Если установлено значение 0, фоновое изображение мастера, установленное с помощью WizardBackImageFile, будет полностью прозрачным. Если установлено значение 255, изображения мастера будут полностью непрозрачными.

Эта функция может пригодиться, если, к примеру, вы хотите использовать одни и те же фоновое изображения как для обычного, так DynamicDark режимов. Например, укажите для WizardBackImageOpacity значение 150 и получите частичную прозрачность.

Пример:
WizardStyle=modern dynamic
WizardBackImageFile=MyBackImage.png
WizardBackImageFileDynamicDark=MyBackImage.png
WizardBackImageOpacity=150
