Препроцессор Inno Setup: GetFileVersionString

Прототип функции

str GetFileVersionString(str FileName)

Описание

Возвращает строку версии файла из ресурса информации о версии указанного файла. Равнозначно вызову GetStringFileInfo(FileName, FILE_VERSION).

Дополнительно смотрите функцию GetVersionNumbersString, которую также можно использовать для получения версии файла. Разница между этими двумя функциями заключается в том, что GetVersionNumbersString получает двоичную версию файла из фиксированного блока ресурсов, а GetFileVersionString получает текстовую версию файла, специфичного для языка.

В случае неудачи функция возвращает значение null (void).

Функция объявлена в файле ISPPBuiltins.iss.
