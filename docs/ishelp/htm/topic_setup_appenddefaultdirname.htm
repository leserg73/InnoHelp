[Setup]: AppendDefaultDirName

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

По умолчанию, когда пользователь выберет папку в диалоговом окне, который отображается при нажатии кнопки Обзор... на странице "Выбор папки установки", программа установки автоматически добавит в новый путь имя папки из директивы DefaultDirName. Например, если директива DefaultDirName имеет значение {autopf}\My Program и пользователь выберет каталог "Z:\", новый путь установки программы будет "Z:\My Program".

Установка для этой директивы значения no отключает добавление папки из параметра DefaultGroupName в новый путь. Кроме того, в диалоговом окне выбора папки появится кнопка Создать папку.
