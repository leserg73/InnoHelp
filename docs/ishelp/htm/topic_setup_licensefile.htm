[Setup]: LicenseFile

Описание

Определяет файл лицензионного соглашения в формате .txt или .rtf (текст с разметкой), который будет показан перед выбором папки установки приложения. При сборке инсталлятора файл документа должен находиться исходном каталоге сценария, в противном случае к этому файлу необходимо указать полный путь или путь с приставкой "compiler:", тогда компилятор будет искать его в собственном каталоге.

Если пользователь выберет язык, для которого указан параметр LicenseFile, эта директива фактически будет проигнорирована. Для получения дополнительной информации обратитесь в разделу справки "Секция [Languages]".

Если используется Unicode файл .txt, он должен быть в кодировке UTF-8 (с маркером BOM или без него) или UTF-16LE.

При использовании .rtf файлов с пользовательскими цветами в тексте обращайте внимание на коэффициенты контрастности, особенно если поддерживается тёмный стиль. Используя инструмент проверки контрастности постарайтесь достичь как минимум уровня AA для обычного текста. Примеры цветов, которые проходят проверку как на светлом, так и на тёмном фоне: красный #eb0000, зеленый #008a00 и синий #6161ff.

Пример:
LicenseFile=license.txt
