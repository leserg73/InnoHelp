[Setup]: UninstallLogging

Допустимые значения:
yes или no

Значение по умолчанию:
no

Описание

При установке значения yes, программа удаления всегда будет создавать файл отчёта, если она запускается из апплета панели управления "Установка и удаление программ". Эквивалентно передаче в командной строке параметра /LOG.

Эта директива не имеет эффекта, если для CreateUninstallRegKey не установлено значение yes.
