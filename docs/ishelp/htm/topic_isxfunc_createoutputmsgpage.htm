Программирование: CreateOutputMsgPage

Прототип

function CreateOutputMsgPage(const AfterID: Integer; const ACaption, ADescription, AMsg: String): TOutputMsgWizardPage;

Описание

Выполняет создание страницы мастера установки, которая содержит только статичный текст.

Параметр AfterID определяет страницу мастера установки, после которой будет размещена (показана) эта страница.

Параметр ACaption определяет текст для заголовка страницы, а параметр ADescription — краткое описание назначения страницы.

Параметр AMsg определяет неизменный текст для отображения.

Пример

var
  Page: TOutputMsgWizardPage;

...

// Создаём страницу
Page := CreateOutputMsgPage(wpWelcome,
  'Информация', 'Пожалуйста, ознакомьтесь с важной информацией.',
  'Бла-бла-бла.');

Смотрите другой пример в файле AllPagesExample.iss из папки "Examples" в каталоге установки Inno Setup.
