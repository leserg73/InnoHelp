[Setup]: EnableDirDoesntExistWarning

Допустимые значения:
yes или no

Значение по умолчанию:
no

Описание

При значении yes программа установки выдаст сообщение, что выбранная пользователем папка не существует.

Как правило, если для этой директивы указывается значение yes, то для DirExistsWarning должно быть задано значение no.
