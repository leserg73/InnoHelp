Препроцессор Inno Setup: ForceDirectories

Прототип функции

int ForceDirectories(str)

Описание

Создаёт все каталоги по указанному пути. Если первый каталог в пути существует, кроме последнего, то будут созданы только те каталоги, которые не существуют. В случае успешного завершения операции возвращает ненулевое значение.
