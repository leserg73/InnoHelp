Программирование: ModifyPifFile

Прототип

function ModifyPifFile(const Filename: String; const CloseOnExit: Boolean): Boolean;

Описание

Выполняет переключение параметра "Закрыть при выходе" в настройках pif-файла, указанного в параметре Filename. В случае успеха возвращает значение True. Параметр CloseOnExit определяет состояние опции: включенное — True или отключенное — False.
