Программирование: ExecAndLogOutputWithNativeSysDir

Прототип

function ExecAndLogOutputWithNativeSysDir(const Filename, Params, WorkingDir: String; const ShowCmd: Integer; const Wait: TExecWait; var ResultCode: Integer; const OnLog: TOnLog): Boolean;

Описание

Используйте эту функцию вместо ExecAndLogOutput для запуска исполняемого файла, расположенного в 64-разрядном системном каталоге, из 32-разрядного установщика.

Эта функция позволяет избежать проблемы, связанной с тем, что функция ExecAndLogOutput в 32-разрядном установщике может запускать такой исполняемый файл только с помощью псевдонима Sysnative, который недоступен для 64-разрядных приложений. Для этого функция временно отключает перенаправление файловой системы WOW64.

Комментарии

В 32-разрядной Windows или в 64-разрядном установщике эта функция является просто псевдонимом для ExecAndLogOutput.
