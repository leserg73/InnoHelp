Программирование: RegQueryStringValue

Прототип

function RegQueryStringValue(const RootKey: Integer; const SubKeyName, ValueName: String; var ResultStr: String): Boolean;

Описание

Запрашивает в указанном разделе реестра значение с типом REG_SZ или REG_EXPAND_SZ и возвращает данные в параметре ResultStr. В случае успеха функция возвращает значение True. Если функция возвращает False, то данные ResultStr остаются без изменений.

Пример

var
  Country: String;
begin
  if RegQueryStringValue(HKEY_CURRENT_USER, 'Control Panel\International',
     'sCountry', Country) then
  begin
    // Успешно получили значение
    MsgBox('Ваша страна: ' + Country, mbInformation, MB_OK);
  end;
end;
