[Setup]: WizardSizePercent

Формат:
a,b, где a – это размер по горизонтали, а b – размер по вертикали.

Допустимые значения:
каждый размер: от 100 до 150

Значение по умолчанию:
120,120, если WizardStyle=modern, иначе 100,100

Описание

Позволяет увеличить стандартный размер всех окон мастера установки без увеличения размера шрифта. К примеру, значение 120 означает увеличение размеров на 20%.

Если в директиве указано только одно значение, то оно будет использовано для изменения размера как по горизонтали, так и по вертикали.

Пример:
WizardSizePercent=120

Чтобы добавить полную поддержку WizardResizable и WizardSizePercent для всех пользовательских элементов управления и страниц мастера, если таковые имеются, используйте свойства Anchors и KeepSizeY. В качестве примера смотрите файл сценария CodeClasses.iss.

Примечание: некоторые окна мастера, такие как "Выбор языка", будут увеличиваться только по горизонтали.
