Программирование: PathEndsWith

Прототип

function PathEndsWith(const S, AEndsWith: String; const IgnoreCase: Boolean): Boolean;

Описание

Возвращает значение True, если строка S заканчивается подстрокой (или равна) AEndsWith. Использует независимое от локали "прямое" (ordinal) сравнение, что важно при сравнении имён файлов и путей. При необходимости может игнорировать регистр символов.
