Программирование: Delete

Прототип

procedure Delete(var S: AnyString; Index, Count: Integer);

Описание

Выполняет удаление заданного количества символов Count из указанной строки S, начиная с позиции Index.

Если значение Index больше, чем длина строки S, то никакие символы не удаляются.

Если для значения Count указано бо́льшее количество символов, чем доступно в строке S с указанной позиции Index, то функция удалит только имеющиеся до конца строки символы.
