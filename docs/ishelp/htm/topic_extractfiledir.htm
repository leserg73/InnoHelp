Препроцессор Inno Setup: ExtractFileDir

Прототип функции

str ExtractFileDir(str PathName)

Описание

Выполняет извлечение пути к папке для заданного файла, исключая завершающий символ обратной косой черты (кроме корневого каталога). Если PathName не содержит путь или его части, то функция возвращает пустую строку.

Функция объявлена в файле ISPPBuiltins.iss.
