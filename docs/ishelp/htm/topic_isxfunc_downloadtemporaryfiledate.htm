Программирование: DownloadTemporaryFileDate

Прототип

function DownloadTemporaryFileDate(const Url): String;

Описание

Возвращает последнюю дату изменения файла по указанному URL-адресу без загрузки самого файла. Если сервер не предоставляет такой информации, функция вернёт значение '' (пустую строку).

В случае ошибки будет вызвано исключение.

Дополнительные сведения смотрите в описании функции DownloadTemporaryFile.
