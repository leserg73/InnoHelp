[Setup]: DiskClusterSize

Значение по умолчанию:
512

Описание

Определяет размер кластера дискового носителя. Компилятору необходимо знать это значение, чтобы оптимально заполнить объём каждого диска.

Эта директива не учитывается, если не включено разделение данных на диски при помощи DiskSpanning в секции [Setup].
