Программирование: StringJoin

Прототип

function StringJoin(const Separator: String; const Values: TArrayOfString): String;

Описание

Объединяет две или более строк, разделённых заданным разделителем.

Пример

var
  S: String;
begin
  S := StringJoin(',', ['1', '2']);
  // S = '1,2'
end;
