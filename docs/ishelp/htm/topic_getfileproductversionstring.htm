Препроцессор Inno Setup: GetFileProductVersionString

Прототип функции

str GetFileProductVersionString(str FileName)

Описание

Возвращает строку версии продукта из ресурса информации о версии указанного файла. Равнозначно вызову GetStringFileInfo(FileName, PRODUCT_VERSION).

В случае неудачи функция возвращает значение null (void).

Функция объявлена в файле ISPPBuiltins.iss.
