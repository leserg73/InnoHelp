Программирование: GetTempDir

Прототип

function GetTempDir: String;

Описание

Возвращает полный путь к временному каталогу, при этом строка пути будет содержать завершающий символ разделения папок (обратная косая черта). Использует функцию API Win32 GetTempPath2, если она доступна, в противном случае – GetTempPath.
