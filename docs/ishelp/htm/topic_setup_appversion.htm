[Setup]: AppVersion

Описание

Эта директива определяет номер версии устанавливаемого приложения. Значение может содержать константы и используется в формировании значения директивы AppVerName, а также добавляется в сведения о приложении для поля "Версия" в апплете Панели управления "Удаление или изменение программы". К тому же, используется для установки значений для ключей MajorVersion и MinorVersion в разделе реестра "Uninstall", когда это возможно.

Эта директива является обязательной и не может быть пустой, если параметр AppVerName не используется.

Пример:
AppVersion=1.5
