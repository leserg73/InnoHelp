Программирование: WizardSelectedTasks

Прототип

function WizardSelectedTasks(const Descriptions: Boolean): String;

Описание

Возвращает разделенный запятыми перечень имен или описаний задач, выбранных пользователем. Если параметр Descriptions имеет значение True, то функция вернёт список описаний, в противном случае — имена.
