Препроцессор Inno Setup: GetSHA1OfString

Прототип функции

str GetSHA1OfString(str)

Описание

Возвращает строковое значение контрольной суммы SHA-1 для заданной строки в кодировке ANSI.

Пример

// Объявляем переменную SHA1 и присваиваем ей значение
// контрольной суммы строки "Test", которое вернёт функция.
#define SHA1 GetSHA1OfString('Test')
// SHA1 = '640ab2bae07bedc4c163f679a746f7ab7fb5d1fa'
