Препроцессор Inno Setup: Pos

Прототип функции

int Pos(str 1, str 2)

Описание

Выполняет поиск подстроки (первый параметр) в другой строке (второй параметр) и возвращает целое значение, которое характеризует начальную позицию (по первому символу) искомой подстроки в пределах заданной строки. Если подстрока не найдена, то функция возвращает нулевое значение.
