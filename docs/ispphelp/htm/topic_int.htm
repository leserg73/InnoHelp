Препроцессор Inno Setup: Int

Прототип функции

int Int(any 1, int? 2)

Описание

Выполняет преобразование строкового выражения, которое задано в качестве первого параметра, в целочисленный тип данных. Если выражение является целым числом, то функция возвращает его в качестве результата. Если выражение не определено (не существует, пустое), то функция возвращает значение 0 (ноль).

Второй параметр является необязательным и позволяет задать значение, которое вернёт функция в случае, если преобразование строкового выражения завершилось неудачей.
