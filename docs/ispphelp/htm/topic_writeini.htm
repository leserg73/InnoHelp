Препроцессор Inno Setup: WriteIni

Прототип функции

void WriteIni(str 1, str 2, str 3, any 4)

Описание

Выполняет запись указанного значения в INI-файл. Первый параметр является именем INI-файла. Второй параметр — это имя секции в INI-файле. Третий параметр — имя ключа в указанной секции. Четвертый параметр должен содержать значение (любого типа), которое необходимо записать в INI-файл.
