Inno Setup Preprocessor: #undef

Синтаксис

undef-directive: (undef | x) [private | protected | public] <ident>

Описание

Выполняет отмену (удаление) объявленной переменной или пользовательской функции. Если область видимости переменной (public, protected, или private) не указана, то ISPP сначала удалит переменную "private", затем "protected" и, наконец, "public".

Примеры

#undef MyVar
#undef MyFunction
#undef public MyVar

См. также:

define
dim
Область видимости переменных
